{1}
[1.1]	na svato nāpi parato na dvābhyāṃ nāpy ahetutaḥ / utpannā jātu vidyante bhāvāḥ kva cana ke cana //
[1.2]	catvāraḥ pratyayā hetur ārambaṇam anantaram / tathaivādhipateyaṃ ca pratyayo nāsti pañcamaḥ //
[1.3]	na hi svabhāvo bhāvānāṃ pratyayādiṣu vidyate / avidyamāne svabhāve parabhāvo na vidyate //
[1.4]	kriyā na pratyayavatī nāpratyayavatī kriyā / pratyayā nākriyāvantaḥ kriyāvantaś ca santy uta //
[1.5]	utpadyate pratītyemān itīme pratyayāḥ kila / yāvan notpadyata ime tāvan nāpratyayāḥ katham //
[1.6]	naivāsato naiva sataḥ pratyayo 'rthasya yujyate / asataḥ pratyayaḥ kasya sataś ca pratyayena kim //
[1.7]	na san nāsan na sad asan dharmo nirvartate yadā / kathaṃ nirvartako hetur evaṃ sati hi yujyate //
[1.8]	anārambaṇa evāyaṃ san dharma upadiśyate / athānārambaṇe dharme kuta ārambaṇaṃ punaḥ //
[1.9]	anutpanneṣu dharmeṣu nirodho nopapadyate / nānantaram ato yuktaṃ niruddhe pratyayaś ca kaḥ //
[1.10]	bhāvānāṃ niḥsvabhāvānāṃ na sattā vidyate yataḥ / satīdam asmin bhavatīty etan naivopapadyate //
[1.11]	na ca vyastasamasteṣu pratyayeṣvasti tat phalam / pratyayebhyaḥ kathaṃ tac ca bhaven na pratyayeṣu yat //
[1.12]	athāsad api tat tebhyaḥ pratyayebhyaḥ pravartate / apratyayebhyo 'pi kasmān nābhipravartate phalam //
[1.13]	phalaṃ ca pratyayamayaṃ pratyayāścāsvayaṃmayāḥ / phalamasvamayebhyo yattatpratyayamayaṃ katham //
[1.14]	tasmān na pratyayamayaṃ nāpratyayamayaṃ phalam / saṃvidyate phalābhāvāt pratyayāpratyayāḥ kutaḥ //

{2}
[2.1]	gataṃ na gamyate tāvad agataṃ naiva gamyate / gatāgatavinirmuktaṃ gamyamānaṃ na gamyate //
[2.2]	ceṣṭā yatra gatis tatra gamyamāne ca sā yataḥ / na gate nāgate ceṣṭā gamyamāne gatis tataḥ //
[2.3]	gamyamānasya gamanaṃ kathaṃ nāmopapatsyate / gamyamānaṃ vigamanaṃ yadā naivopapadyate //
[2.4]	gamyamānasya gamanaṃ yasya tasya prasajyate / ṛte gater gamyamānaṃ gamyamānaṃ hi gamyate //
[2.5]	gamyamānasya gamane prasaktaṃ gamanadvayam / yena tad gamyamānaṃ ca yac cātra gamanaṃ punaḥ //
[2.6]	dvau gantārau prasajyete prasakte gamanadvaye / gantāraṃ hi tiraskṛtya gamanaṃ nopapadyate //
[2.7]	gantāraṃ cet tiraskṛtya gamanaṃ nopapadyate / gamane 'sati gantātha kuta eva bhaviṣyati //
[2.8]	gantā na gacchati tāvad agantā naiva gacchati / anyo gantur agantuś ca kas tṛtīyo 'tha gacchati //
[2.9]	gantā tāvad gacchatīti katham evopapatsyate / gamanena vinā gantā yadā naivopapadyate //
[2.10]	pakṣo gantā gacchatīti yasya tasya prasajyate / gamanena vinā gantā gantur gamanam icchataḥ //
[2.11]	gamane dve prasajyete gantā yadyuta gacchati / ganteti cājyate yena gantā san yac ca gacchati //
[2.12]	gate nārabhyate gantuṃ gantuṃ nārabhyate 'gate / nārabhyate gamyamāne gantum ārabhyate kuha //
[2.13]	na pūrvaṃ gamanārambhād gamyamānaṃ na vā gatam / yatrārabhyeta gamanam agate gamanaṃ kutaḥ //
[2.14]	gataṃ kiṃ gamyamānaṃ kim agataṃ kiṃ vikalpyate / adṛśyamāna ārambhe gamanasyaiva sarvathā //
[2.15]	gantā na tiṣṭhati tāvad agantā naiva tiṣṭhati / anyo gantur agantuś ca kas tṛtīyo 'tha tiṣṭhati //
[2.16]	gantā tāvat tiṣṭhatīti katham evopapatsyate / gamanena vinā gantā yadā naivopapadyate //
[2.17]	na tiṣṭhati gamyamānān na gatān nāgatād api / gamanaṃ saṃpravṛttiś ca nivṛttiś ca gateḥ samā //
[2.18]	yad eva gamanaṃ gantā sa eveti na yujyate / anya eva punar gantā gater iti na yujyate //
[2.19]	yad eva gamanaṃ gantā sa eva hi bhaved yadi / ekībhāvaḥ prasajyeta kartuḥ karmaṇa eva ca //
[2.20]	anya eva punar gantā gater yadi vikalpyate / gamanaṃ syād ṛte gantur gantā syād gamanād ṛte //
[2.21]	ekībhāvena vā siddhir nānābhāvena vā yayoḥ / na vidyate tayoḥ siddhiḥ kathaṃ nu khalu vidyate //
[2.22]	gatyā yayājyate gantā gatiṃ tāṃ sa na gacchati / yasmān na gatipūrvo 'sti kaścid kiṃcid dhi gacchati //
[2.23]	gatyā yayājyate gantā tato 'nyāṃ sa na gacchati / gatī dve nopapadyete yasmād eke tu gantari //
[2.24]	sadbhūto gamanaṃ gantā triprakāraṃ na gacchati / nāsadbhūto 'pi gamanaṃ triprakāraṃ sa gacchati //
[2.25]	gamanaṃ sadasadbhūtaḥ triprakāraṃ na gacchati / tasmād gatiś ca gantā ca gantavyaṃ ca na vidyate //

[3.1]	darśanaṃ śravaṇaṃ ghrāṇaṃ rasanaṃ sparśanaṃ manaḥ / indriyāṇi ṣaḍ eteṣāṃ draṣṭavyādīni gocaraḥ //
[3.2]	svam ātmānaṃ darśanaṃ hi tat tam eva na paśyati / na paśyati yad ātmānaṃ kathaṃ drakṣyati tat parān //
[3.3]	na paryāpto 'gnidṛṣṭānto darśanasya prasiddhaye / sadarśanaḥ sa pratyukto gamyamānagatāgataiḥ //
[3.4]	nāpaśyamānaṃ bhavati yadā kiṃ cana darśanam / darśanaṃ paśyatīty evaṃ katham etat tu yujyate //
[3.5]	paśyati darśanaṃ naiva naiva paśyaty adarśanam / vyākhyāto darśanenaiva draṣṭā cāpy avagamyatām //
[3.6]	draṣṭā nāsty atiraskṛtya tiraskṛtya ca darśanam / draṣṭavyaṃ darśanaṃ caiva draṣṭary asati te kutaḥ //
[3.7]	pratītya mātāpitarau yathoktaḥ putrasaṃbhavaḥ / cakṣūrūpe pratītyaivamukto vijñānasaṃbhavaḥ //
[3.8]	draṣṭavyadarśanābhāvād vijñānādicatuṣṭayam / nāstīty upādānādīni bhaviṣyanti punaḥ katham //
[3.9]	vyākhyātaṃ śravaṇaṃ ghrāṇaṃ rasanaṃ sparśanaṃ manaḥ / darśanenaiva jānīyāc chrotṛśrotavyakādi ca //

{4}
[4.1]	rūpakāraṇanirmuktaṃ na rūpam upalabhyate / rūpeṇāpi na nirmuktaṃ dṛśyate rūpakāraṇam //
[4.2]	rūpakāraṇanirmukte rūpe rūpaṃ prasajyate / āhetukaṃ na cāsty arthaḥ kaścid āhetukaḥ kva cit //
[4.3]	rūpeṇa tu vinirmuktaṃ yadi syād rūpakāraṇam / akāryakaṃ kāraṇaṃ syād nāsty akāryaṃ ca kāraṇam //
[4.4]	rūpe saty eva rūpasya kāraṇaṃ nopapadyate / rūpe 'saty eva rūpasya kāraṇaṃ nopapadyate //
[4.5]	niṣkāraṇaṃ punā rūpaṃ naiva naivopapadyate / tasmād rūpagatān kāṃścin na vikalpān vikalpayet //
[4.6]	na kāraṇasya sadṛśaṃ kāryam ity upapadyate / na kāraṇasyāsadṛśaṃ kāryam ity upapadyate //
[4.7]	vedanācittasaṃjñānāṃ saṃskārāṇāṃ ca sarvaśaḥ / sarveṣām eva bhāvānāṃ rūpeṇaiva samaḥ kramaḥ //
[4.8]	vigrahe yaḥ parīhāraṃ kṛte śūnyatayā vadet / sarvaṃ tasyāparihṛtaṃ samaṃ sādhyena jāyate //
[4.9]	vyākhyāne ya upālambhaṃ kṛte śūnyatayā vadet / sarvaṃ tasyānupālabdhaṃ samaṃ sādhyena jāyate //

{5}
[5.1]	nākāśaṃ vidyate kiṃcit pūrvam ākāśalakṣaṇāt / alakṣaṇaṃ prasajyeta syāt pūrvaṃ yadi lakṣaṇāt //
[5.2]	alakṣaṇo na kaścic ca bhāvaḥ saṃvidyate kva cit / asaty alakṣaṇe bhāve kramatāṃ kuha lakṣaṇam //
[5.3]	nālakṣaṇe lakṣaṇasya pravṛttir na salakṣaṇe / salakṣaṇālakṣaṇābhyāṃ nāpy anyatra pravartate //
[5.4]	lakṣaṇāsaṃpravṛttau ca na lakṣyam upapadyate / lakṣyasyānupapattau ca lakṣaṇasyāpy asaṃbhavaḥ //
[5.5]	tasmān na vidyate lakṣyaṃ lakṣaṇaṃ naiva vidyate / lakṣyalakṣaṇanirmukto naiva bhāvo 'pi vidyate //
[5.6]	avidyamāne bhāve ca kasyābhāvo bhaviṣyati / bhāvābhāvavidharmā ca bhāvābhāvāv avaiti kaḥ //
[5.7]	tasmānna bhāvo nābhāvo na lakṣyaṃ nāpi lakṣaṇam / ākāśam ākāśasamā dhātavaḥ pañca ye 'pare //
[5.8]	astitvaṃ ye tu paśyanti nāstitvaṃ cālpabuddhayaḥ / bhāvānāṃ te na paśyanti draṣṭavyopaśamaṃ śivam //

{6}
[6.1]	rāgād yadi bhavet pūrvaṃ rakto rāgatiraskṛtaḥ / taṃ pratītya bhaved rāgo rakte rāgo bhavet sati //
[6.2]	rakte 'sati punā rāgaḥ kuta eva bhaviṣyati / sati vāsati vā rāge rakte 'py eṣa samaḥ kramaḥ //
[6.3]	sahaiva punar udbhūtir na yuktā rāgaraktayoḥ / bhavetāṃ rāgaraktau hi nirapekṣau parasparam //
[6.4]	naikatve sahabhāvo 'sti na tenaiva hi tat saha / pṛthaktve sahabhāvo 'tha kuta eva bhaviṣyati //
[6.5]	ekatve sahabhāvaś cet syāt sahāyaṃ vināpi saḥ / pṛthaktve sahabhāvaś cet syāt sahāyaṃ vināpi saḥ //
[6.6]	pṛthaktve sahabhāvaś ca yadi kiṃ rāgaraktayoḥ / siddhaḥ pṛthakpṛthagbhāvaḥ sahabhāvo yatas tayoḥ //
[6.7]	siddhaḥ pṛthakpṛthagbhāvo yadi vā rāgaraktayoḥ / sahabhāvaṃ kimartham tu parikalpayase tayoḥ //
[6.8]	pṛthag na sidhyatīty evaṃ sahabhāvaṃ vikāṅkṣasi / sahabhāvaprasiddhyarthaṃ pṛthaktvaṃ bhūya icchasi //
[6.9]	pṛthagbhāvāprasiddheś ca sahabhāvo na sidhyati / katamasmin pṛthagbhāve sahabhāvaṃ satīcchasi //
[6.10]	evaṃ raktena rāgasya siddhir na saha nāsaha / rāgavat sarvadharmāṇāṃ siddhir na saha nāsaha //

{7}
[7.1]	yadi saṃskṛta utpādas tatra yuktā trilakṣaṇī / athāsaṃskṛta utpādaḥ kathaṃ saṃskṛtalakṣaṇam //
[7.2]	utpādādyās trayo vyastā nālaṃ lakṣaṇakarmaṇi / saṃskṛtasya samastāḥ syur ekatra katham ekadā //
[7.3]	utpādasthitibhaṅgānām anyat saṃskṛtalakṣaṇam / asti ced anavasthaivaṃ nāsti cet te na saṃskṛtāḥ //
[7.4]	utpādotpāda utpādo mūlotpādasya kevalam / utpādotpādam utpādo maulo janayate punaḥ //
[7.5]	utpādotpāda utpādo mūlotpādasya te yadi / maulenājanitas taṃ te sa kathaṃ janayiṣyati //
[7.6]	sa te maulena janito maulaṃ janayate yadi / maulaḥ sa tenājanitas tam utpādayate katham //
[7.7]	ayam utpādyamānas te kāmam utpādayed imam / yadīmam utpādayitum ajātaḥ śaknuyād ayam //
[7.8]	pradīpaḥ svaparātmānau saṃprakāśayate yathā / utpādaḥ svaparātmānāv ubhāv utpādayet tathā //
[7.9]	pradīpe nāndhakāro 'sti yatra cāsau pratiṣṭhitaḥ / kiṃ prakāśayate dīpaḥ prakāśo hi tamovadhaḥ //
[7.10]	katham utpadyamānena pradīpena tamo hatam / notpadyamāno hi tamaḥ pradīpaḥ prāpnute yadā //
[7.11]	aprāpyaiva pradīpena yadi vā nihataṃ tamaḥ / ihasthaḥ sarvalokasthaṃ sa tamo nihaniṣyati //
[7.12]	pradīpaḥ svaparātmānau saṃprakāśayate yadi / tamo 'pi svaparātmānau chādayiṣyaty asaṃśayam //
[7.13]	anutpanno 'yam utpādaḥ svātmānaṃ janayet katham / athotpanno janayate jāte kiṃ janyate punaḥ //
[7.14]	notpadyamānaṃ notpannaṃ nānutpannaṃ kathaṃ cana / utpadyate tad ākhyātaṃ gamyamānagatāgataiḥ //
[7.15]	utpadyamānam utpattāv idaṃ na kramate yadā / katham utpadyamānaṃ tu pratītyotpattim ucyate //
[7.16]	pratītya yad yad bhavati tat tac chāntaṃ svabhāvataḥ / tasmād utpadyamānaṃ ca śāntam utpattir eva ca //
[7.17]	yadi kaścid anutpanno bhāvaḥ saṃvidyate kva cit / utpadyeta sa kiṃ tasmin bhāva utpadyate 'sati //
[7.18]	utpadyamānam utpādo yadi cotpādayaty ayam / utpādayet tam utpādam utpādaḥ katamaḥ punaḥ //
[7.19]	anya utpādayaty enaṃ yady utpādo 'navasthitiḥ / athānutpāda utpannaḥ sarvam utpadyatāṃ tathā //
[7.20]	sataś ca tāvad utpattir asataś ca na yujyate / na sataś cāsataś ceti pūrvam evopapāditam //
[7.21]	nirudhyamānasyotpattir na bhāvasyopapadyate / yaś cānirudhyamānas tu sa bhāvo nopapadyate //
[7.22]	nāsthitas tiṣṭhate bhāvaḥ sthito bhāvo na tiṣṭhati / na tiṣṭhate tiṣṭhamānaḥ ko 'nutpannaś ca tiṣṭhati //
[7.23]	sthitir nirudhyamānasya na bhāvasyopapadyate / yaś cānirudhyamānas tu sa bhāvo nopapadyate //
[7.24]	jarāmaraṇadharmeṣu sarvabhāveṣu sarvadā / tiṣṭhanti katame bhāvā ye jarāmaraṇaṃ vinā //
[7.25]	sthityānyayā sthiteḥ sthānaṃ tayaiva ca na yujyate / utpādasya yathotpādo nātmanā na parātmanā //
[7.26]	nirudhyate nāniruddhaṃ na niruddhaṃ nirudhyate / tathā nirudhyamānaṃ ca kim ajātaṃ nirudhyate //
[7.27]	sthitasya tāvad bhāvasya nirodho nopapadyate / nāsthitasyāpi bhāvasya nirodha upapadyate //
[7.28]	tayaivāvasthayāvasthā na hi saiva nirudhyate / anyayāvasthayāvasthā na cānyaiva nirudhyate //
[7.29]	yadaiva sarvadharmāṇām utpādo nopapadyate / tadaivaṃ sarvadharmāṇāṃ nirodho nopapadyate //
[7.30]	sataś ca tāvad bhāvasya nirodho nopapadyate / ekatve na hi bhāvaś ca nābhāvaś copapadyate //
[7.31]	asato 'pi na bhāvasya nirodha upapadyate / na dvitīyasya śirasaś chedanaṃ vidyate yathā //
[7.32]	na svātmanā nirodho 'sti nirodho na parātmanā / utpādasya yathotpādo nātmanā na parātmanā //
[7.33]	utpādasthitibhaṅgānām asiddher nāsti saṃskṛtam / saṃskṛtasyāprasiddhau ca kathaṃ setsyaty asaṃskṛtam //
[7.34]	yathā māyā yathā svapno gandharvanagaraṃ yathā / tathotpādas tathā sthānaṃ tathā bhaṅga udāhṛtam //

[8.8]	satā ca kriyate nāsan nāsatā kriyate ca sat / kartrā sarve prasajyante doṣās tatra ta eva hi //
[8.9]	nāsadbhūtaṃ na sadbhūtaḥ sadasadbhūtam eva vā / karoti kārakaḥ karma pūrvoktair eva hetubhiḥ //
[8.10]	nāsadbhūto 'pi sadbhūtaṃ sadasadbhūtam eva vā / karoti kārakaḥ karma pūrvoktair eva hetubhiḥ //
[8.11]	karoti sadasadbhūto na san nāsac ca kārakaḥ / karma tat tu vijānīyāt pūrvoktair eva hetubhiḥ //
[8.12]	pratītya kārakaḥ karma taṃ pratītya ca kārakam / karma pravartate nānyat paśyāmaḥ siddhikāraṇam //
[8.13]	evaṃ vidyād upādānaṃ vyutsargād iti karmaṇaḥ / kartuś ca karmakartṛbhyāṃ śeṣān bhāvān vibhāvayet //

[15.1]	na saṃbhavaḥ svabhāvasya yuktaḥ pratyayahetubhiḥ / hetupratyayasaṃbhūtaḥ svabhāvaḥ kṛtako bhavet //

[25.17]	paraṃ nirodhād bhagavān bhavatīty eva nājyate / na bhavaty ubhayaṃ ceti nobhayaṃ ceti nājyate //
[25.18]	tiṣṭhamāno 'pi bhagavān bhavatīty eva nājyate / na bhavaty ubhayaṃ ceti nobhayaṃ ceti nājyate //
[25.19]	na saṃsārasya nirvāṇāt kiṃcid asti viśeṣaṇam / na nirvāṇasya saṃsārāt kiṃcid asti viśeṣaṇam //
[25.20]	nirvāṇasya ca yā koṭiḥ koṭiḥ saṃsaraṇasya ca / na tayor antaraṃ kiṃcit susūkṣmam api vidyate //
[25.21]	paraṃ nirodhād antādyāḥ śāśvatādyāś ca dṛṣṭayaḥ / nirvāṇam aparāntaṃ ca pūrvāntaṃ ca samāśritāḥ //
[25.22]	śūnyeṣu sarvadharmeṣu kim anantaṃ kim antavat / kim anantam antavac ca nānantaṃ nāntavac ca kim //
[25.23]	kiṃ tad eva kim anyat kiṃ śāśvataṃ kim aśāśvatam / aśāśvataṃ śāśvataṃ ca kiṃ vā nobhayam apy atha //
[25.24]	sarvopalambhopaśamaḥ prapañcopaśamaḥ śivaḥ / na kva cit kasyacit kaścid dharmo buddhena deśitaḥ //

{26}
[26.1]	punarbhavāya saṃskārān avidyānivṛtas tridhā / abhisaṃskurute yāṃs tair gatiṃ gacchati karmabhiḥ //
[26.2]	vijñānaṃ saṃniviśate saṃskārapratyayaṃ gatau / saṃniviṣṭe 'tha vijñāne nāmarūpaṃ niṣicyate //
[26.3]	niṣikte nāmarūpe tu ṣaḍāyatanasaṃbhavaḥ / ṣaḍāyatanam āgamya saṃsparśaḥ saṃpravartate //
[26.4]	cakṣuḥ pratītya rūpaṃ ca samanvāhāram eva ca / nāmarūpaṃ pratītyaivaṃ vijñānaṃ saṃpravartate //
[26.5]	saṃnipātas trayāṇāṃ yo rūpavijñānacakṣuṣām / sparśaḥ sa tasmāt sparśāc ca vedanā saṃpravartate //
[26.6]	vedanāpratyayā tṛṣṇā vedanārthaṃ hi tṛṣyate / tṛṣyamāṇa upādānam upādatte caturvidham //
[26.7]	upādāne sati bhava upādātuḥ pravartate / syād dhi yady anupādāno mucyeta na bhaved bhavaḥ //
[26.8]	pañca skandhāḥ sa ca bhavo bhavāj jātiḥ pravartate / jarāmaraṇaduḥkhādi śokāḥ saparidevanāḥ //
[26.9]	daurmanasyam upāyāsā jāter etat pravartate / kevalasyaivam etasya duḥkhaskandhasya saṃbhavaḥ //
[26.10]	saṃsāramūlaṃ saṃskārān avidvān saṃskaroty ataḥ / avidvān kārakas tasmān na vidvāṃs tattvadarśanāt //
[26.11]	avidyāyāṃ niruddhāyāṃ saṃskārāṇām asaṃbhavaḥ / avidyāyā nirodhas tu jñānasyāsyaiva bhāvanāt //
[26.12]	tasya tasya nirodhena tat tan nābhipravartate / duḥkhaskandhaḥ kevalo 'yam evaṃ samyag nirudhyate //

{27}
[27.1]	abhūm atītam adhvānaṃ nābhūvam iti dṛṣṭayaḥ / yās tāḥ śāśvatalokādyāḥ pūrvāntaṃ samupāśritāḥ //
[27.2]	dṛṣṭayo na bhaviṣyāmi kim anyo 'nāgate 'dhvani / bhaviṣyāmīti cāntādyā aparāntaṃ samāśritāḥ //
[27.3]	abhūm atītam adhvānam ity etan nopapadyate / yo hi janmasu pūrveṣu sa eva na bhavaty ayam //
[27.4]	sa evātmeti tu bhaved upādānaṃ viśiṣyate / upādānavinirmukta ātmā te katamaḥ punaḥ //
[27.5]	upādānavinirmukto nāsty ātmeti kṛte sati / syād upādānam evātmā nāsti cātmeti vaḥ punaḥ //
[27.6]	na copādānam evātmā vyeti tat samudeti ca / kathaṃ hi nāmopādānam upādātā bhaviṣyati //
[27.7]	anyaḥ punar upādānād ātmā naivopapadyate / gṛhyeta hy anupādāno yady anyo na ca gṛhyate //
[27.8]	evaṃ nānya upādānān na copādānam eva saḥ / ātmā nāsty anupādāno nāpi nāsty eṣa niścayaḥ //
[27.9]	nābhūm atītam adhvānam ity etan nopapadyate / yo hi janmasu pūrveṣu tato 'nyo na bhavaty ayam //
[27.10]	yadi hy ayaṃ bhaved anyaḥ pratyākhyāyāpi taṃ bhavet / tathaiva ca sa saṃtiṣṭhet tatra jāyeta cāmṛtaḥ //
[27.11]	ucchedaḥ karmaṇāṃ nāśaḥ kṛtam anyena karma ca / pratisaṃvedayed anya evam ādi prasajyate //
[27.12]	nāpy abhūtvā samudbhūto doṣo hy atra prasajyate / kṛtako vā bhaved ātmā saṃbhūto vāpy ahetukaḥ //
[27.13]	evaṃ dṛṣtir atīte yā nābhūm aham abhūm aham / ubhayaṃ nobhayaṃ ceti naiṣā samupapadyate //
[27.14]	adhvany anāgate kiṃ nu bhaviṣyāmīti darśanam / na bhaviṣyāmi cety etad atītenādhvanā samam //
[27.15]	sa devaḥ sa manuṣyaś ced evaṃ bhavati śāśvatam / anutpannaś ca devaḥ syāj jāyate na hi śāśvatam //
[27.16]	devād anyo manuṣyaś ced aśāśvatam ato bhavet / devād anyo manuṣyaś cet saṃtatir nopapadyate //
[27.17]	divyo yady ekadeśaḥ syād ekadeśaś ca mānuṣaḥ / aśāśvataṃ śāśvataṃ ca bhavet tac ca na yujyate //
[27.18]	aśāśvataṃ śāśvataṃ ca prasiddham ubhayaṃ yadi / siddhe na śāśvataṃ kāmaṃ naivāśāśvatam ity api //
[27.19]	kutaścid āgataḥ kaścit kiṃcid gacchet punaḥ kva cit / yadi tasmād anādis tu saṃsāraḥ syān na cāsti saḥ //
[27.20]	nāsti cec chāśvataḥ kaścit ko bhaviṣyaty aśāśvataḥ / śāśvato 'śāśvataś cāpi dvābhyām ābhyāṃ tiraskṛtaḥ //
[27.21]	antavān yadi lokaḥ syāt paralokaḥ kathaṃ bhavet / athāpy anantavāl lokaḥ paralokaḥ kathaṃ bhavet //
[27.22]	skandhānām eṣa saṃtāno yasmād dīpārciṣām iva / tasmān nānantavattvaṃ ca nāntavattvaṃ ca yujyate //
[27.23]	pūrve yadi ca bhajyerann utpadyeran na cāpy amī / skandhāḥ skandhān pratītyemān atha loko 'ntavān bhavet //
[27.24]	pūrve yadi na bhajyerann utpadyeran na cāpyamī / skandhāḥ skandhān pratītyemāl loko 'nanto bhaved atha //
[27.25]	antavān ekadeśaś ced ekadeśas tv anantavān / syād antavān anantaś ca lokas tac ca na yujyate //
[27.26]	kathaṃ tāvad upādātur ekadeśo vinaṅkṣyate / na naṅkṣyate caikadeśa evaṃ caitan na yujyate //
[27.27]	upādānaikadeśaś ca kathaṃ nāma vinaṅkṣyate / na naṅkṣyate caikadeśo naitad apy upapadyate //
[27.28]	antavac cāpy anantaṃ ca prasiddham ubhayaṃ yadi / siddhe naivāntavat kāmaṃ naivānantavad ity api //
[27.29]	atha vā sarvabhāvānāṃ śūnyatvāc chāśvatādayaḥ / kva kasya katamāḥ kasmāt saṃbhaviṣyanti dṛṣṭayaḥ //
[27.30]	sarvadṛṣṭiprahāṇāya yaḥ saddharmam adeśayat / anukampām upādāya taṃ namasyāmi gautamam //
