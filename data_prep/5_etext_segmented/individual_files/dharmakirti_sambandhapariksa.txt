[01_03]	pāratantryam hi sambandhaḥ siddhe kā paratantra tā tasmāt sarvasya bhāvasya sambandhaḥ na asti tattvataḥ rūpa śleṣaḥ hi sambandhaḥ dvi tve sa ca katham bhavet tasmāt prakṛti bhinnānām sambandhaḥ na asti tattvataḥ para apekṣāḥ hi sambandhaḥ saḥ asan katham apekṣate san ca sarva nirāśaṃsaḥ bhāvaḥ katham apekṣate 
[04_06]	dvayoḥ eka abhisambandhāt sambandhaḥ yadi tad dvayoḥ kaḥ sambandhaḥ anavasthā ca na sambandha matiḥ tathā tau ca bhāvau tad anyaḥ ca sarve te sva ātmani sthitāḥ iti amiśrāḥ svayam bhāvāḥ tān miśrayati kalpanā tām eva ca anurundhānaiḥ kriyā kāraka vācinaḥ bhāva bheda pratīti artham saṃyojyante abhidhāyakāḥ 
[07_09]	kārya kāraṇa bhāvaḥ api tayoḥ asahabhāvataḥ prasiddhyati katham dviṣṭhaḥ adviṣṭhe sambandha tā katham krameṇa bhāvaḥ ekatra vartamāṇaḥ anya nispṛhaḥ tad abhāve api tad bhāvāt sambandhaḥ na eka vṛttimān yadi apekṣya tayoḥ ekam anyatra asau pravartate upakārī hi apekṣyaḥ syāt katham ca upakaroti asan 
[10_13]	yadi eka artha abhisambandhāt kārya kāraṇa tā tayoḥ prāptāḥ dvi tva ādi sambandhāt savya itara viśāṇayoḥ dviṣṭhaḥ hi kaścid sambandhaḥ na atas anyat tasya lakṣaṇam bhāva abhāva upadhiḥ yogaḥ kārya kāraṇa tā yadi yoga upādhī na tau eva kārya kāraṇa tā atra kim bhedāt ced nanu ayam śabdaḥ niyoktāram samāśritaḥ paśyan ekam adṛṣṭasya darśane tad adarśane apaśyat kāryam anveti vinā vyākhyātṛbhiḥ janaḥ 
[14_17]	darśana adarśane muktvā kārya buddheḥ asambhavāt kārya ādi śrutiḥ api atra lāghava artham niveśitā tad bhāva abhāvāt tad kārya gatiḥ yā api anuvarṇyate saṃketa viṣaya ākhyā sā sāsnādeḥ go gatiḥ yathā bhāve bhāvini tad bhāvaḥ bhāvaḥ eva ca bhāvitā prasiddhe hetu phalate pratyakṣa anupalambhataḥ etāvat mātra tattva arthāḥ kārya kāraṇa gocarāḥ vikalpāḥ darśayanti arthān mithyā arthāḥ ghaṭitān iva 
[18_21]	bhinne kā ghaṭanā bhinne kārya kāraṇa tā api kā bhāve hi anyasya viśliṣṭau śliṣṭau syātām katham ca tau saṃyogi sama vāyi ādi sarvam etena cintitam anyonya anupakārāt ca na sambandhī ca tādṛśaḥ janane api hi kāryasya kenacid samavāyinā samavāyī tadā na asau na tatas ati prasaṅgataḥ tayoḥ anupakāre api samavāye paratra vā sambandhaḥ yadi viśvam syāt samavāyi parasparam
