STPradīpa_0.18^1	 tat ca aṇimā mahimā ca eva garimā laghimā tathā puruṣasya tathā ātmānam prakāśya vinivartate prakṛtiḥ iti nanu ekādaśakaḥ indriya gaṇaḥ pañca tanmātrakam ca iti dvividhaḥ pravartate sargaḥ a haṅkārāt tathā ca ekasmāt ahaṅkārāt kāraṇāt katham prakāśa jaḍa ātmakaḥ vilakṣaṇaḥ sargaḥ iti yat uktam tat na sundaram prakāśa lāghavābhyām ekādaśakaḥ indriya gaṇaḥ sāttvikāt ahaṅkārāt bhavati bhūtādeḥ tu ahaṅkārāt tāmasāt tanmātra gaṇaḥ yadi api ekaḥ ahaṅkāraḥ tathā api guṇa udbhava abhibhavābhyām bhinnam kāryam karoti nanu yadi sattva tamobhyām eva sargaḥ kṛtam tarhi rajasā akiñci kareṇa iti ced na taijasāt ubhayam iti mūlakaraḥ taijasāt ubhayam gaṇa dvayam bhavati sarga dvayam sattva tamasoḥ iti yadi api rajasaḥ na kārya antaram asti tathā api sattva tamasī svayam akriye samarthe api na kāryam kurutaḥ rajasā cālite tu tad prasādāt prāpta kriye sva kārya utpādane samarthe bhavataḥ iti rājasāt ubhayam 
STPradīpa_0.18^2	 yathā āhuḥ ubhayasmin= api kārye sattva tamasoḥ kriyā utpādana dvāreṇa asti rajasaḥ kāraṇa tvam iti na vyartham rajaḥ iti indriya vicāraḥ tu tattva pradīpe kṛtaḥ iti vistara bhayāt na iha pratanyate mahat ahaṅkāra manāṃsi antar indriyāṇi tāni sva lakṣaṇāni svam asādhāraṇam lakṣaṇam yeṣām tāni sva lakṣaṇāni tat ca mahataḥ adhyavasāyaḥ ahaṅkārasya abhimānaḥ saṅkalpaḥ manasaḥ vṛttiḥ vyāpāraḥ eteṣām vṛtti dvaividhyam sādhāraṇa tvāsā dhāraṇa tvābhyām āha sā eṣā bhavati asāmānyāḥ iti eṣā uktā asādhāraṇī vṛttiḥ sādhāraṇī vṛttiḥ tu karaṇānām prāṇa ādyāḥ vāyavaḥ pañca karaṇa trayasya pañca vāyavaḥ vṛttiḥ jīvanam tad bhāve bhāvāt abhāve ca abhāvāt tatra prāṇaḥ nāsā agra hṛd nābhi pāda aṅguṣṭha vṛttiḥ apānaḥ kṛkāṭikā pṛṣṭha pārśva pāyu upastha vṛttiḥ samānaḥ hṛd nābhi sarva sandhi vṛttiḥ udānaḥ hṛd kaṇṭha tāla mūrdha bhrū madhya vṛttiḥ vyānaḥ tvac vṛttiḥ iti pañca vāyavaḥ 
STPradīpa_0.19	 avidyā pañcavidhā tathā hi avidyā asmitā rāga dveṣa abhiniveśāḥ yathāsaṃkhyam tamaḥ moha mahā moha tāmisra andhatāmisra saṃjñakā ca iti pañca parvā pratyaya sarga tad bhedāḥ tu ati vistara bhayāt santaḥ api na nidarśyante pratyaya sargaḥ buddhi sargaḥ uktañcaḥ eṣa pratyaya sargaḥ viparyaya aśakti tuṣṭi siddhi ākhyaḥ pratīyate anena iti pratyayaḥ buddhiḥ tasya sargaḥ pratyaya sargaḥ daiva sargaḥ tu mūlakāreṇa eva spaṣṭaḥ niruktaḥ sa ca aṣṭa vikalpaḥ daivaḥ sargaḥ brāhma prājāpatya aindra paitrya gāndharvayākṣarākṣa sa paiśācaḥ iti aṣṭa vikalpaḥ daiva sargaḥ tiryagyonayaḥ ca pañcavidhāḥ bhavanti paśu mṛga pakṣi sarīsṛpa sthāvarāḥ mānuṣaḥ ca ekavidhaḥ brāhmaṇa tva ādyavantara jāti bheda avivakṣayā saṃsthānasya caturṣu api aviśeṣāt iti 
STPradīpa_0.20^1	 nanu prādhānikaḥ ayam sargaḥ pradhānam ca jaḍam katham sva artham para artham vā pravṛttim kartum arhati tasmāt kenacid cetanena prakṛti adhiṣṭhātrā bhavitavyam na ca kṣetrajñāḥ tathā teṣām prakṛti svarūpa anabhijña tvāt tasmāt asti prakṛteḥ adhiṣṭhātā kaścid sarvajñaḥ saḥ eva īśvaraḥ iti ced na acetanam api prayojana vaśāt pravartamānam dṛṣṭam yathā vatsa vivṛddhi artham acetanam api kṣīram pravartate tathā jaḍā api prakṛtiḥ puruṣa vimokṣāya pravartiṣyate kiñca abhoktuḥ īśvarasya pravṛttau na karuṇām vinā anyat prayojanam paśyāmaḥ sā ca nirupādhi para duḥkha prahāṇa icchā sṛṣṭeḥ pūrvam prāṇi abhāvāt eva tad sambandhi duḥkha abhāvāt kasya nivṛttau pravṛttiḥ syāt īśvarasya na ca sarga anantaram īśvara pravṛttiḥ sargasya jāta tvāt eva na īśvara prayojanam paśyāmaḥ 
STPradīpa_0.20^2	 na ca duḥkha nivṛttiḥ prayojanam tad artham tad pravṛttau jagati kaḥ api duḥkhī na syāt uktañca prekṣāvataḥ pravṛtteḥ sva artha kāruṇyābhyām vyāpta tvāt te ca jagat sargāt vyāvartamāne prekṣāvat pravṛtti pūrvaka tvam api nivartayataḥ na hi āpta sakala īpsitasya bhagavataḥ jagat sṛjataḥ kim api abhilaṣitam bhavati na api kāruṇyāt asya sarge pravṛttiḥ prāk sargāt jīvānām indriya śarīra viṣaya anutpattau duḥkha abhāvena kasya prahāṇa icchā kāruṇyena sṛṣṭiḥ sṛṣṭyā ca kāruṇyam iti paraspara āśrayaḥ api ca karuṇayā pravṛttaḥ sukhinaḥ eva jantūn sṛjet na daridrān na ca karma vaicitryāt jagat vaicitryam tarhi kṛtam īśvareṇa tatas eva jagat vaicitrya upapatteḥ pravṛttiḥ tu pārārthya prayojana vaśāt jaḍasya api dṛśyate vatsa vivṛddhi nimittam dugdhasya iva 
STPradīpa_0.21	 nanu bhavatu prakṛteḥ pravartakaḥ puruṣa arthaḥ nivartakam na tu kam api paśyāma iti ced na yathā nartakī pariṣadbhyaḥ nṛtyam darśayitvā svayam eva nivartate na tu tasyāḥ sva nivṛttau nivartaka antara apekṣā tathā śabda ādi ātmakam sva svarūpam puruṣāt bhedena prakāśya sādhita puruṣa prayojanā svayam eva nivartate prakṛtiḥ 
STPradīpa_0.22	 yathā āhuḥ sāṃkhya ācāryāḥ raṅgasya darśayitvā nivartate yathā nartakī nṛtyāt puruṣasya tathā ātmānam prakāśya vinivartate prakṛtiḥ nanu yathā darśita nṛtyā nartakī draṣṭṛ kautuhalāt punar pravartate tathā prakṛtiḥ api puruṣāya ātmānam darśayitvā nivṛttā api punar ced pravartete tadā muktānām api punar saṃsāra prāptiḥ syāt iti ced na yathā para puruṣa darśana asahā sūrya darśana varjitā api kula vadhūḥ pramādāt vigalita aṅga paṭā ced avalokyate para puruṣeṇa tadā asau tathā pravartate apramattām yathā enām punar puruṣa antarāni na paśyanti tathā prakṛtiḥ kula vadhūtaḥ api adhikā dṛṣṭā biṃbakena punar na darśanam ārgam eti 
STPradīpa_0.23	 uktam ca prakṛteḥ sukumārataram na kiñcid asti iti me matiḥ bhavati yā dṛṣṭā asmi iti punar na darśanam upaiti puruṣasya iti dharmaḥ jñānam virāga aiśvaryam iti sāttvikāḥ buddhi dharmāḥ adharma ajñāna avairāgya anaiśvarya abhidhānāḥ catvāraḥ tāmasāḥ atra jñāna varjitaiḥ saptabhiḥ prakṛtiḥ badhnāti ātmānam viveka khyātyā prakṛti puruṣa anya tā jñānena sā eva vimocayati
