SKBh_65	 jñāne puruṣaḥ kim karoti tena nivṛtta prasavām artha vaśāt sapta rūpa vinivṛttām prakṛtim paśyati puruṣaḥ prekṣaka vat avasthitaḥ svasthaḥ tena viśuddhena kevala jñānena puruṣaḥ prakṛtim paśyati prekṣaka vat prekṣakeṇa tulyam avasthitaḥ svasthaḥ yathā raṃga prekṣakaḥ avasthitaḥ narttakīm paśyatiḥ svasthaḥ svasmin= tiṣṭhati svasthaḥ sva sthāna sthitaḥ kathaṃbhūtām prakṛtim nivṛtta prasavām nivṛttaḥ buddhi ahaṃkāra kāryya anartha vaśāt sapta rūpa vinivṛttām nivarttita ubhaya puruṣa prayojana vaśa ādyaiḥ saptabhiḥ rūpaiḥ dharma ādibhiḥ ātmānam badhnāti tebhyaḥ saptabhyaḥ rūpebhyaḥ vinivṛttim prakṛtim paśyati 
SKBh_66	 kim ca dṛṣṭā mayā iti upekṣakaḥ ekaḥ dṛṣṭā aham iti uparamati anyā sati saṃyoge api tayoḥ prayojanam na asti sargasya raṃga sthaḥ iti yathā raṃga sthaḥ iti evam upekṣakaḥ ekaḥ kevalaḥ śuddhaḥ puruṣaḥ tena aham dṛṣṭā iti kṛtvā uparatā nivṛtā ekā ekā eva prakṛtiḥ trailokyasya api pradhāna kāraṇa bhūtā na dvitīyā prakṛtiḥ asti mūrtti bhede jāti bhedāt evam prakṛti puruṣayoḥ nirvṛttau= api vyāpaka tvāt saṃyogaḥ asti na tu saṃyogāt kutas sargaḥ bhavati sati saṃyoge api tayoḥ prakṛti puruṣayoḥ sarva gata tvāt sati api saṃyoge prayojanam na asti sargasya sṛṣṭeḥ caritārtha tvāt prakṛteḥ dvividham prayojanam śabda viṣaya upalabdhiḥ guṇa puruṣa antara upalabdhiḥ ca ubhayatra api caritārtha tvāt sargasya na asti prayojanam yaḥ punar sargaḥ iti yathā dāna grahaṇa nimittaḥ uttamarṇa adhamarṇayoḥ dravya viśuddhau sati api saṃyoge na kaścid artha sambandhaḥ bhavati evam prakṛti puruṣayoḥ api na asti prayojanam iti 
SKBh_67	 yadi puruṣasya utpanne jñāne mokṣaḥ bhāti tatas mama kasmāt na bhavati iti atas ucyate samyak jñāna adhigamāt dharma ādīnām akāraṇa prāptau tiṣṭhati saṃskāra vaśāt cakra bhrama vat dhṛta śarīraḥ yadi api pañcaviṃśati tattva jñānam samyak jñānam bhavati tathā api saṃskāra vaśāt dhṛta śarīraḥ yogī tiṣṭhati katham cakra bhrama vat cakra bhrameṇa tulyam yathā kulālaḥ cakram bhramayitvā ghaṭam karoti mṛd piṇḍam cakram āropya punar kṛtvā ghaṭam paryāmuñcati cakram bhramati eva saṃskāra vaśāt evam samyak jñāna adhigamāt utpanna samyak jñānasya dharma ādīnām akāraṇa prāptau etāni sapta rūpāṇi bandhana bhūtāni samyak jñānena dagdhāni yathā na agninā dagdhāni bījāni prarohaṇa samarthāni evam etāni dharma ādīni bandhanāni na samarthāni dharma ādīnām akāraṇa prāptau saṃskāra vaśāt dhṛta śarīrāḥ tiṣṭhati jñānāt varttamāna dharma adharma kṣayaḥ kasmāt na bhavati varttamāna tvāt eva kṣaṇa antare kṣayam apyeti jñānam tu anāgata karma dahati varttamāna śarīreṇa ca yat karoti tat api iti vihita anuṣṭhāna karaṇāt iti saṃskāra kṣayāt śarīra pāte mokṣaḥ
