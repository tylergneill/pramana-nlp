MM_7.3	 vivāda adhyāsitāḥ śabdāḥ ākāśa samavāyena eva samavāyinaḥ samavāyi tvāt śabda ākāśa vat na ca samavāyasya ekatve vastu saṅkara prasaṅgaḥ tasya samasta viśva asaṃkara tvena eva parikalpita tvāt na hi tasya apratyakṣasya pūrvam svabhāvaḥ nirdhāritaḥ ekasya bhinna svabhāva tve anyaḥ iti ced na citra rūpavat tathāvidhasya eka svabhāvāt evam ca kalpanā lāghavāt samavāyasya niyatvam kvacid pareṇa abhyupagamāt tasmāt tad samavāyasya nitya tvam eka tvasya sādhita tvāt samavāyaḥ asmad ādi pratyakṣaḥ na bhavati asmāt ādi pratyakṣa apratyakṣa sambandha tvāt ghaṭa ākāśa sambandha vat nitya sambandhaḥ samavāyaḥ iha pratyaya viṣayībhāvaḥ 
MM_8.1–8.2	 yadi na asti iti pratyaya vedyamānaḥ api tvayā na iṣṭaḥ iti vaktum na ucitam kevala jñānam ca asmābhiḥ abhyupagatam iti tat api vikalpya dūṣaṇīyam patana ādi kāryam pratibandhaka abhāva utpādaka kāraṇāt eva utpādyate kārya tvāt dāha ādi vat aghaṭam bhū talam iti vyavahāraḥ bhū tala ati rikta viṣayaḥ bhū tala mātra vyavahāra atirikta tve sati bhū tala vyavahāra tvāt iti mānam bhū talam iti vyavahāreṇa anaikāntikam tathā api abhāvaḥ abhyupagantavyaḥ kevala jñāna abhāve api abhāva vyahāra darśanāt bhavati hi ghaṭa tad rūpayoḥ saha upalambhe api rūpam ghaṭaḥ na bhavati iti bhinnaḥ ca utpāda vyavahāraḥ na ca tatra kevala jñānam asti ādhāra agrahaṇe api sarveṣām antarāle āloka abhāva vyavahārāt kara parāmarśeṇa andhakāre andhasya api mūrta abhāva vyavahāra darśanāt kiñca cakṣuḥ cakṣuḥ grāhya bhāva vyatirikta grāhakam indriya tvāt 
MM_8.3	 sa caturvidhaḥ prāc abhāva pradhvaṃsa abhāva anyonya abhāva atyanta abhāva bhedāt pratiyogi bhedāt bhinna tvam abhāvasya tādātmya saṃsṛṣṭa pūrva apara saṃsṛṣṭānām pratiyogi tvāt anyathā pratibandhaka abhāvasya kāraṇa tvam durghaṭam atyanta abhāvasya api itaretara abhāve antarbhāvāt tasya ca sati api pratibandhe bhāvāt vyabhicāraḥ samavāya anyaḥ asamavāyi abhāvaḥ 
MM_9.1	 śaktiḥ api padārtha antaram asti iti ced na tad sadbhāve pramāṇa abhāvāt sphoṭa anupapattiḥ iti ced na pratibandha abhāvena api upapatteḥ tatra ca prāc abhāva ādi vikalpāḥ yogyatā anupalabdhi sāmānyena parihartavyāḥ kārya lakṣaṇa abhyupagamena uttambhaka mantrasya sphoṭa utpattau= api na aniyata hetutvam brīhīn prokṣati iti dvitīyāyāḥ ca na tad samaveta saṃskāra vācaka tvam kriyā janya phala bhāgi tvam ca na tad samavāyinam ākṣipati kintu tādarthya mātram anyathā upayuktānām prastara ādīnām homena utpādyamānaḥ saṃskāraḥ na teṣām syāt teṣām vinaṣṭa tvāt na ca tatra saṃskārya tvam na asti bhūta bhāvi upayogam ca dravyam saṃskāram arhati iti nyāyāt tatra ca nirāśrayasya saṃskārasya avasthāna anupapatteḥ ātma āśraya tvam kalpanīyam tadvat itaratrāpi bhavatu kaḥ doṣaḥ 
MM_9.2	 ghaṭam aham jānāmi iti atra prābhākarāṇām naye jñāna janita phalasya ghaṭe kasyacid abhāvāt tatra katham karma tvam iti cintyam atas eva bhāṭṭaiḥ jñāta tā abhyupagamyate iti ced teṣām api atīta anāgateṣu kā vārttā tatra api jāteḥ anuvartamāna tvāt adoṣaḥ iti ced katham tarhi vyaktau jñāta tvam tayoḥ abhedāt iti ced na bhedasya api vidyamāna tvāt bhede api adoṣaḥ iti ced na rasanena rasa jñāna mātreṇa eva tad āśrayasya api jñāta tva prasaṅgāt sattā jñāna tva mātreṇa eva jāti andhasya api rūpa jñāna prasaṅgaḥ jñāta tā anabhyupagamāt katham jñānasya viṣaya niyamaḥ iti ced jñāta tāyāḥ eva katham āśraya niyamaḥ kāraṇamahimnā iti ced samaḥ samādhiḥ 
MM_9.3–9.4	 etena viśiṣṭa tā api nirākṛtāḥ tad utpādaka sāmagryāḥ eva vijātīya jñāna utpattau= api upapatteḥ na ca viṣaya vailakṣaṇya abhāve vaijātyam durghaṭam anubhava smṛtyoḥ avaijātya prasaṅgāt adhika viṣaya tve smṛteḥ api pramāṇa tva prasaṅgāt anyāyāḥ viśiṣṭa tāyāḥ atīta anāgateṣu avasthanam durghaṭam āśraya abhāvāt etena viṣayībhāva rūpaḥ sambandhaḥ nirastaḥ tad niyāmakena eva jñāna arthayoḥ niyama upapatteḥ evam sānnidhya ādayaḥ api nirākartavyāḥ 
MM_9.5	 astu tarhi sādṛśyam padārtha antaram iti ced na māna abhāvāt na ca vyavahāraḥ mānam tasya sāmānya ādi sambandha tvena api upapatteḥ katham tarhi sāmānye sādṛśya vyavahāraḥ kutas śyena citam cinvītaḥ iti atra atideśanam tatra api vyakti sādṛśyasya eva pratīteḥ vyakteḥ ca śabda artha tvāt śyena tva eka artha samavetadvitradharmavataḥ anuṣṭhānasya sambhavāt saṃketa grahasya aupādhika śabda vat upapatteḥ 
MM_9.6^1	 evam pradhāna ādayaḥ api nirākāryāḥ sādhaka abhāvāt na ca vivāda padam sākṣāt paramparayā vā prakṛti janyam kāryatvāt iti avītam iti vācyam vyapti asiddheḥ na ca sukha duḥkha mohāḥ kāraṇa pūrvakāḥ kāryatvāt iti sāmpratam saṃyoga ādiṣu vyabhicārāt na ca sat utpatti paryālocanayā tad siddhaḥ tatra eva vivādāt na ca tad artham upādīyamāna tvāt tatra sattvam nimitta kāraṇe vyabhicārāt na ca sati kāraṇa vyāpāraḥ asārthaka tvāt iti jāti niyama jñena vyavasthā ghaṭate 
MM_9.6^2	 yena eva pramāṇena ghaṭa bhāva siddhiḥ tena eva pramāṇena yogyā anupalabdhyā paṭa utpatteḥ pūrvam tantuṣu api paṭa abhāva siddhiḥ ayogya tvena anupalabdhi śaṅkāyām itaratra api sambhāvanām kaḥ nivārayet kārya apekṣayā caramasahakāri kāraṇāt utpattiḥ iti na anavasthā dūṣaṇam mūla kṣateḥ abhāvāt na api punarukta tva virodhaḥ ghaṭa ādibhiḥ tasyāḥ ca utpatteḥ abhyupagamāt abhivyakte tu sarvāsām abhivyaktīnām sadātana tvāt dṛṣṭa adṛṣṭa vyāsthā dauḥsthyam kim bahunā paraspara virodhe hi na prakāra antara sthitiḥ iti nyāyāt padārtha antarāṇām na sambhavaḥ sapta lakṣaṇānām paraspara virodhāt iti 
MM_10.1	 tatra hi dravya ādinām heya upādeya svarūpeṇa jñāne sati saṃsārasya duḥkha bhūyaḥ tvam paśyataḥ sukham api duḥkha anuṣaṅgāt duḥkha pakṣe nikṣiptam upaniṣadā ca jñāta jīva ātma brahmaṇaḥ vaiśeṣika ukta dṛśā bhagavat bhakti manaskasya sat guru kaṭākṣa pāta kṣīṇa antarāyasya parameśvara sākṣātkāra kutūhalinaḥ ahorātram nididhi āsanam ācarataḥ ananta kāla krameṇa paripakva jñānasya prakṛṣṭa yoga ja dharma śālinaḥ kathañcid sākṣāt kṛta parameśvarasya pravṛddha pratipakṣa niravakāśīkṛta mithyājñānasya kāraṇa abhāvāt apagata rāga dveṣa pravṛtti sambhāvanasya aśeṣa duḥkha abhāvāt svena rūpeṇa avasthānam bhavati sa ca ayam mokṣaḥ iti manīṣibhiḥ udāhriyate 
MM_10.2	 na ca sukhasya api abhāvāt apravṛttiḥ bhavatām api sukhasya apravṛtti kara tvāt na hi tat sādhyam nitya tva vyāghātāt na hi siddhārthe kaścid prayatate iti bhavatām api avidyā nivṛttiḥ eva kṛti sādhyā tatas ca tat eva prayojanam yathā āha yam artham adhikṛtya pravartate tat prayojanam ānandam brahmaṇaḥ rūpam tat ca mokṣe abhivyajyate iti santoṣa ja sukha bhiprāyeṇa gauṇyā vṛttyā duḥkha abhāva abhiprāyeṇa vā kathaṃcid guṇa vādaḥ iti aśarīram vāva santam na priya apriye spṛśataḥ iti vacanāt na ca sāṃsārika sukha abhiprāyeṇa tat sāṃsārika sukhasya api brahma sukha an atirikta tvāt svayam prakāśa tve ca svasya mukti saṃsārayoḥ aviśeṣa prasaṅgāt 
MM_10.3	 atra kecid duḥkha parihāra artham jñāna sukhayoḥ pūrvam viṣaya viṣayi bhāvaḥ anupapannaḥ paścāt utpādyaḥ iti brūyuḥ tat asāmpratam tad sadbhāve pramāṇa abhāvāt yasya loke aprasiddhiḥ tasya prayojaka tvam iti api durghaṭam eva na iha jñānam asti viṣaya viṣayi bhāvaḥ na asti iti dṛṣṭa pūrvam jñānasya vākyena utpādaka sāmagryāḥ eva vā tad utpādaka tvāt pradhāna ādi nirāsena eva tat ukta mokṣasya api bhaṅgaḥ veditavyaḥ 
MM_10.4	 vimatam kadācid viśeṣa guṇa rahitam vijñāna asamavāyi kāraṇa ādhāra tvāt anitya viśeṣa guṇa ādhāra tvāt manaḥ vat vyoma vat ca na ca jīvāḥ nitya jñānāḥ ātma vat īśvara vat iti sāmpratam īśvara svarūpasya eva upādheḥ piṇḍavyabhicāra abhāvāt na ca nitya viśeṣa guṇa ādhārāḥ iti sādhyarthaḥ apārthivat vyaṇukā ārambhaka tve sati viśeṣa guṇavat tva upādheḥ viśeṣaṇa abhāvāt bhavatām punar viśeṣa guṇa rahita tvāt sukha tvam nitya samavetam tad jāti tvāt iti tu tad gata nikarṣa tvena vyabhicarata ati alam ati prasaṅgena namaḥ tripathak ākrānta jaṭā jūṭāya śambhave namaḥ te jagat utpatti sthiti dhvaṃsa vidhāyine
