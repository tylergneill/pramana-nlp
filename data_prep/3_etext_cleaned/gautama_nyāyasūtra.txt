[5.1.1]	sādharmyavaidharmyotkarṣāpakarṣavarṇyāvarṇyavikalpasādhyaprāptyaprāptiprasaṅgapratidṛṣṭāntānutpattisaṃśayaprakaraṇāhetvarthāpattyaviśeṣopapattyupalabdhyanupalabdhyanityanityakāryasamāḥ //
[5.1.2]	sādharmyavaidharmyābhyām upasaṃhāre taddharmaviparyayopapatteḥ sādharmyavaidharmyasamau //
[5.1.3]	gotvād gosiddhivat tatsiddhiḥ //
[5.1.4]	sādhyadṛṣṭāntayor dharmavikalpād ubhayasādhyatvāc cotkarṣāpakarṣavarṇyāvarṇyavikalpasādhyasamāḥ //
[5.1.5]	kiñcitsādharmyād upasaṃhārasiddher vaidharmyād apratiṣedhaḥ //
[5.1.6]	sādhyātideśāc ca dṛṣṭāntopapatteḥ //
[5.1.7]	prāpya sādhyam aprāpya vā hetoḥ prāptyāviśiṣṭatvād aprāptyāsādhakatvāc ca prāptyaprāptisamau //
[5.1.8]	ghaṭādiniṣpattidarśanāt pīḍane cābhicārād apratiṣedhaḥ //
[5.1.9]	dṛṣṭāntasya kāraṇānapadeśāt pratyavasthānāc ca pratidṛṣṭāntena prasaṅgapratidṛṣṭāntasamau //
[5.1.10]	pradīpopādānaprasaṅgavinivṛttivat tadvinivṛttiḥ //
[5.1.11]	pratidṛṣṭāntahetutve ca nāhetur dṛṣṭāntaḥ //
[5.1.12]	prāgutpatteḥ kāraṇābhāvād anutpattisamaḥ //
[5.1.13]	tathābhāvād utpannasya kāraṇopapatter na kāraṇapratiṣedhaḥ //
[5.1.14]	sāmānyadṛṣṭāntayor aindriyakatve samāne nityānityasādharmyāt saṃśayasamaḥ //
[5.1.15]	sādharmyāt saṃśaye na saṃśayo vaidharmyād ubhayathā vā saṃśaye 'tyantasaṃśayaprasaṅgo nityatvānabhyupagamāc ca sāmānyasyāpratiṣedhaḥ //
[5.1.16]	ubhayasādharmyāt prakriyāsiddheḥ prakaraṇasamaḥ //
[5.1.17]	pratipakṣāt prakaraṇasiddheḥ pratiṣedhānupapattiḥ pratipakṣopapatteḥ //
[5.1.18]	traikālyāsiddher hetor ahetusamaḥ //
[5.1.19]	na hetutaḥ sādhyasiddhes traikālyāsiddhiḥ //
[5.1.20]	pratiṣedānupapatteś ca pratiṣeddhavyāpratiṣedhaḥ //
[5.1.21]	arthāpattitaḥ pratipakṣasiddher arthāpattisamaḥ //
[5.1.22]	anuktasyārthāpatteḥ pakṣahāner upapattir anuktatvād anaikānatikatvāc cārthāpatteḥ //
[5.1.23]	ekadharmopapatter aviśeṣe sarvāviśeṣaprasaṅgāt sadbhāvaupapatter aviśeṣasamaḥ //
[5.1.24]	kvacit taddharmopapatteḥ kvacic cānupapatteḥ pratiṣedhābhāvaḥ //
[5.1.25]	ubhayakāraṇopapatter upapattisamaḥ //
[5.1.26]	upapattikāraṇābhyanujñānād apratiṣedhaḥ //
[5.1.27]	nirdiṣṭakāraṇābhāve 'py upalambhād upalabdhisamaḥ //
[5.1.28]	kāraṇāntarād api taddharmopapatter apratiṣedhaḥ //
[5.1.29]	tadanupalabdher anupalambhād abhāvasiddhau tadviparītopapatter anupalabdhisamaḥ //
[5.1.30]	anupalambhātmakatvād anupalabdher ahetuḥ //
[5.1.31]	jñānavikalpānāṃ ca bhāvābhāvasaṃvedanād adhyātmam //
[5.1.32]	sādharmyāt tulyadharmopapatteḥ sarvānityatvaprasaṅgād anityasamaḥ //
[5.1.33]	sādharmyād asiddheḥ pratiṣedhāsiddhiḥ pratiṣedhyasādharmyāt //
[5.1.34]	dṛṣṭānte ca sādhyasādhanabhāvena prajñātasya dharmasya hetutvāt tasya cobhayathābhāvān nāviśeṣaḥ //
[5.1.35]	nityam anityabhāvād anitye nityatvopapatter nityasamaḥ //
[5.1.36]	pratiṣedhye nityam anityabhāvād anitye 'nityatvopapatteḥ pratiṣedhābhāvaḥ //
[5.1.37]	prayatnakāryānekatvāt kāryasamaḥ //
[5.1.38]	kāryānyatve prayatnāhetutvam anupalabdhikāraṇopapatteḥ //
[5.1.39]	pratiṣedhe 'pi samāno doṣaḥ //
[5.1.40]	sarvatraivam //
[5.1.41]	pratiṣedhavipratiṣedhe pratiṣedhadoṣavad doṣaḥ //
[5.1.42]	pratiṣedhaṃ sadoṣam abhyupetya pratiṣedhavipratiṣedhe samāno doṣaprasaṅgo matānujñā //
[5.1.43]	svapakṣalakṣaṇāpekṣopapattyupasaṃhāre hetunirdeśe parapakṣadoṣābhyupagamāt samāno doṣaḥ //

{5.2}

[5.2.1]	pratijñāhāniḥ pratijñāntaraṃ pratijñāvirodhaḥ pratijñāsannyāso hetvantaram arthāntaraṃ nirarthakam avijñātārtham apārthakam aprāptakālaṃ nyūnam adhikaṃ punaruktam ananubhāṣaṇam ajñānam apratibhā vikṣepo matānujñā paryanuyojyopekṣaṇaṃ niranuyojyānuyogo 'pasiddhānto hetvābhāsāś ca nigrahasthānāni //
[5.2.2]	pratidṛṣṭāntadharmābhyanujñā svadṛṣṭānte pratijñāhāniḥ //
[5.2.3]	pratijñātārthapratiṣedhe dharmavikalpāt tadarthanirdeśaḥ pratijñāntaram //
[5.2.4]	pratijñāhetvor virodhaḥ pratijñāvirodhaḥ //
[5.2.5]	pakṣapratiṣedhe pratijñātārthāpanayanaṃ pratijñāsannyāsaḥ //
[5.2.6]	aviśeṣokte hetau pratiṣiddhe viśeṣam icchato hetvantaram //
[5.2.7]	prakṛtād arthād apratisambaddhārtham arthāntaram //
[5.2.8]	varṇakramanirdeśavan nirarthakam //
[5.2.9]	pariṣatprativādibhyāṃ trir abhihitam apy avijñātam avijñātārtham //
[5.2.10]	paurvāparyāyogād apratisambaddhārtham apārthakam //
[5.2.11]	avayavaviparyāsavacanam aprāptakālam //
[5.2.12]	hīnam anyatamenāpy avayavena nyūnam //
[5.2.13]	hetūdāharaṇādhikam adhikam //
[5.2.14]	śabdārthayoḥ punar vacanaṃ punaruktam anyatrānuvādāt //
[5.2.15]	arthād āpannasya svaśabdena punarvacanam //
[5.2.16]	vijñātasya pariṣadā, trir abhihitasyāpy apratyuccāraṇam ananubhāṣaṇam //
[5.2.17]	avijñātaṃ cājñānam //
[5.2.18]	uttarasyāpratipattir apratibhā //
[5.2.19]	kāryavyāsaṅgāt kathāvicchedo vikṣepaḥ //
[5.2.20]	svapakṣe doṣābhyupagamāt parapakṣe doṣaprasaṅgo matānujñā //
[5.2.21]	nigrahasthānaprāptasyānigrahaḥ paryanuyojyopekṣaṇam //
[5.2.22]	anigrahasthāne nigrahasthānābhiyogo niranuyojyānuyogaḥ //
[5.2.23]	siddhāntam abhyupetyāniyamāt kathāprasaṅgo 'pasiddhāntaḥ //
[5.2.24]	hetvābhāsāś ca yathoktāḥ //
