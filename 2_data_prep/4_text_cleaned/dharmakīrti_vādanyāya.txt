[01,01]

namaḥ samantabhadrāya.

[01,02]

nyāyavādinam api vādeṣu asad-vyavasthā-upanyāsaiḥ śaṭhā nigṛhṇanti, tan-niṣedha-artham idam ārabhyate.

[01,04]

asādhana-aṅga-vacanam adoṣa-udbhāvanaṃ dvayoḥ / nigrahasthānam, anyat tu na yuktam iti na iṣyate //1//

<1>

<nigrahasthānalakṣaṇa>

{111.121_0}

[01,06]

iṣṭasya arthasya siddhiḥ sādhanam, tasya nirvartakam aṅgam, tasya avacanam tasya aṅgasya anuccāraṇaṃ vādino nigraha-adhikaraṇam, tad abhyupagamya apratibhayā tūṣṇīṃbhāvāt, sādhana-aṅgasya asamarthād vā.

[01,10]

trividham eva hi liṅgam apratyakṣasya siddher aṅgam, svabhāvaḥ kāryam anupalabdhaś ca. tasya samarthanaṃ sādhyena vyāptiṃ prasādhya dharmiṇi bhāva-sādhanam,

{111.121a}

[01,13]

yathā yat sat kṛtakam vā, tat sarvam anityam, yathā ghaṭa-ādiḥ, san kṛtako vā śabda iti. atra api na kaścit krama- niyamaḥ, iṣṭa-artha-siddher ubhayatra aviśeṣāt. dharmiṇi prāk sattvaṃ prasādhya paścād api vyāptiḥ prasādhyata eva yathā san śabdaḥ kṛtako vā, yaś ca evam, sa sarvo anityaḥ, yathā ghaṭa-ādir iti.

[02,01]

atra vyāpti-sādhanaṃ viparyaye bādhaka-pramāṇa-upadarśanam.

[02,02]

yadi na sarvaṃ sat kṛtakaṃ vā pratikṣaṇa-vināśi syāt, akṣaṇikasya kramayaugapadyābhyām arthakriyā-ayogād arthakriyā-sāmarthya lakṣaṇato nivṛttam ity asad eva syāt.

[02,05]

sarva-sāmarthya-upākhyā viraha-lakṣaṇaṃ hi nirupākhyam iti.

[02,06]

evaṃ sādhanasya sādhya-viparyaye bādhaka-pramāṇa-anupadarśane virodha-abhāvād asya viparyaye vṛtter adarśane api san kṛtako vā syān nityaś ca ity anivṛttir eva śaṅkāyāḥ. tato vyatirekasya sandehād anaikāntikaḥ syād hetvābhāsaḥ.

[02,10]

na apy adarśana-mātrād vyāvṛttiḥ, viprakṛṣṭeṣv asarva-darśino adarśanasya abhāva-asādhanāt, arvāgdarśanena satām api keṣāṃcid arthānām adarśanāt.

[02,13]

bādhakaṃ punaḥ pramāṇam, yatra kramayaugapadya-ayogaḥ, na tasya kvacit sāmarthyam, asti ca akṣaṇike sa iti pravartamānam asāmarthyam asal-lakṣaṇam ākarṣati. tena yat sat kṛtakaṃ vā tad anityam eva iti sidhyati, tāvatā ca sādhana dharma-mātra-anvayaḥ sādhya-dharmasya, svabhāvahetu- lakṣaṇaṃ ca siddhaṃ bhavati.

[02,19]

atra apy adarśanam apramāṇayataḥ kramayaugapadya-ayogasya eva asāmarthyena vyāpty-asiddheḥ pūrvakasya api hetor avyāptiḥ. iha api punaḥ sādhana-upakrame anavasthā-prasaṅga iti cet. na, abhāva sādhanasya adarśanasya apratiṣedhāt. yad adarśanaṃ viparyayaṃ sādhayati hetoḥ sādhya-viparyaye, tad asya viruddha-pratyupasthāpanād bādhakaṃ pramāṇam ucyate. evaṃ hi sa hetuḥ sādhya-abhāve asan sidhyet, yadi tatra pramāṇavatā sva-viruddhena bādhyeta. anyathā tatra asya bādhaka-asiddhau saṃśayo durnivāraḥ. na ca sarva-anupalabdhir bhāvasya bādhikā. tatra sāmarthyaṃ krama-akrama yogena vyāptaṃ siddham, prakāra-antara-abhāvāt. tena vyāpaka- anupalabdhir akṣaṇike sāmarthyaṃ bādhata iti kramayaugapadya-ayogasya sāmarthya-abhāvena vyāpti-siddher na anavasthā-prasaṅgaḥ.

[03,14]

evaṃ svabhāvahetu-prayogeṣu samarthitaṃ sādhana-aṅgaṃ bhavati. tasya asamarthanaṃ sādhana-aṅga-avacanaṃ tad-vādinaḥ parājaya-sthānam ārabdha-artha-aprasādhanāt, vastutaḥ samarthasya hetor upādāne api sāmarthya-apratipādanāt.

{111.121b}

[03,18]

kāryahetāv api sādhana-aṅgasya samarthanam, yat kārya- liṅgaṃ kāraṇasya sādhanāya upādīyate, tasya tena saha kāryakāraṇabhāva- prasādhanaṃ bhāva-abhāva-sādhana-pramāṇābhyām.

[04,02]

yathā idam asmin sati bhavati. satsv api tad-anyeṣu samartheṣu tad-hetuṣu tad-abhāve na bhavati iti. evaṃ hy asya asandigdhaṃ tat-kāryatvaṃ samarthitaṃ bhavati.

[04,05]

anyathā kevalaṃ tad-abhāve na bhavati ity upadarśane anyasya api tatra abhāve sandigdham asya sāmarthyam. anyat tatra samartham, tad-abhāvāt tan na bhūtam. etan-nivṛttau punar nivṛttir yadṛcchā-saṃvādaḥ, mātṛ-vivāha-ucita-deśa-janmaḥ piṇḍa kharjūrasya deśa-antareṣu mātṛ-vivāha-abhāve abhāva-vat.

[04,11]

evaṃ samarthitaṃ tat tat-kāryaṃ sidhyati. siddhaṃ sat sva-sambhavena tat-sambhavaṃ sādhayati. kāryasya kāraṇa- avyabhicārāt. avyabhicāre ca sva-kāraṇaiḥ sarva-kāryāṇāṃ sadṛśo nyāyaḥ. evam asamarthanaṃ kāryahetāv api sādhana- aṅga-avacanaṃ tad-vādinaḥ parājaya-sthānam, asamarthite tasmin kāryatva-asiddher  artha-antarasya tadbhāva-apratibaddha- svabhāvasya bhāve tadbhāva-niyama-abhāvād ārabdha-artha- asiddheḥ, vastutaḥ kāryasya apy upādāne tad-apratipādanāt.

{111.121c1}

[04,20]

anupalabdhāv api pratipattur upalabdhi-lakṣaṇa-prāptasya anupalabdhi-sādhanaṃ samarthanam, tādṛśyā eva anupalabdher asad-vyavahāra-siddheḥ, anupalabdhi-lakṣaṇa-prāptasya pratipattṛ-pratyakṣa-upalabdhi-nivṛttāv apy abhāva-asiddheḥ.

[05,01]

tatra upalabdhi-lakṣaṇa-prāptiḥ svabhāva-viśeṣaḥ kāraṇa-antara- sākalyaṃ ca. svabhāva-viśeṣo yan na trividhena viprakarṣeṇa viprakṛṣṭam, yad anātma-rūpa-pratibhāsa-vivekena pratipattṛ-pratyakṣa-pratibhāsa-rūpam. tādṛśaḥ satsv anyeṣu upalambha-pratyayeṣu tathā anupalabdho asad-vyavahāra- viṣayaḥ. tato anyathā sati liṅge saṃśayaḥ.

{111.121c2}

[05,07]

atra api sarvam evaṃvidham asad-vyavahāra-viṣaya iti vyāptiḥ, kasyacid asato abhyupagame tal-lakṣaṇa-aviśeṣāt; na hy evaṃvidhasya asattvān abhyupagame anyatra tasya yogaḥ.

[05,10]

na hy evaṃvidhasya sataḥ satsv anyeṣu upalambha-kāraṇeṣv anupalabdhiḥ. anupalabhyamāṇaṃ tv īdṛśaṃ na asti ity etāvanmātra- nimitto ayam asad-vyavahāraḥ, anyasya tan-nimittasya abhāvāt.

[05,14]

sarva-sāmarthya-viveko nimittam iti cet, evam etat tasya eva sarva-sāmarthya-vivekina evaṃ pratītiḥ, anyasya tat-pratipatty- upāya-abhāvāt. tat-pratipattau ca satyām asad-vyavahāra iti idaṃ tan-nimittam ucyate.

[05,18]

buddhi-vyapadeśa-arthakriyābhyaḥ sad-vyavahāro viparyaye ca asad-vyavahāra iti cet, bhavati buddher yathokta-pratibhāsāyāḥ sad-vyavahāraḥ, viparyaye asad-vyavahāraḥ.

[05,21]

pratyakṣa-aviṣaye tu syāl liṅgajāyā api kutaścit sad-vyavahāraḥ.

[05,22]

asad-vyavahāras tu tad-viparyaye anaikāntikaḥ, viprakṛṣṭe arthe pratipattṛ-pratyakṣasya anyasya vā pramāṇasya nivṛttāv api saṃśayāt.

[06,01]

na ca sarve buddhi-vyapadeśās tad-bheda-abhedau vā vastu- sattāṃ vastu-bheda-abheda-sattāṃ vā sādhayanti, asatsv api kathaṃcid atīta-anāgata-ādiṣu nānā-eka-arthakriyākāriṣu vā artheṣu tad-bhāva sthāpanāya nānā-ekātma-abhāve api nānā-eka-rūpāṇāṃ vṛtteḥ, rājā mahā-sammataḥ prabhavo rājavaṃśasya, śaṅkhaś cakravarttī mahā-sammata-nirmitasya yūpasya utthāpayitā, śaśa-viṣāṇam, rūpaṃ sanidarśanaṃ sapratigham, ghaṭaś ca iti. na hi sanidarśana-ādi śabdā nānā-vastu-viṣayā ekatra upasaṃhārāt.

[06,10]

nānā-viṣayatve apy ekatra upasaṃhāras tan-nimittānān tatra samavāyād iti cet, āyāse vata ayaṃ tapasvī padārthaḥ patito aneka-sambandhinam upakṛtya-anekaṃ śabdam ātmani tebhyaḥ samāśaṃsan. sa yaiḥ śakti-bhedair anekaṃ sambandhinam upakaroti tair eva anekaṃ śabdaṃ kiṃ na utthāpayati.

[06,15]

evaṃ hy anena paraṃpara-anusāra-pariśramaḥ parihṛto bhavati nānā-śabda-utthāpana asāmarthye nānā-sambandhy- upakāro api mā bhūt, anupakāre hi teṣāṃ tat-sambandhitā api na sidhyati.

[06,19]

ghaṭa ity api ca rūpa-ādaya eva bahava eka-arthakriyākāriṇa eka-śabda-vācyā bhavantu, kim artha-antara-kalpanayā.

[06,21]

bahavo api hy eka-artha-kāriṇo bhaveyuś cakṣur-ādi-vat. tat-sāmarthya- sthāpanāya tatra eka-śabda-niyogo api syād iti yuktaṃ paśyāmaḥ. na ca niḥprayojanā lokasya artheṣu śabda-yojanā.

[06,24]

tatra ye arthāḥ saha pṛthag vā eka-prayojanās teṣāṃ tadbhāva- sthāpanāya hy eka-śabdo niyujyate yadi, kiṃ syāt. tad-arthakriyā- śakti-sthāpanāya niyuktasya samudāya-śabdasya eka- vacana-virodho api na asty eva. sahitānāṃ sā śaktir ekā na pratyekam iti samudāya-śabda ekasmin samudāye vācye eka-vacanaṃ ghaṭa iti. jāti śabdeṣv arthānāṃ pratyekaṃ sahitānāṃ ca śakter nānā-ekā ca śaktir iti, nānā-eka-śakti-vivakṣāyāṃ bahu-vacanam eka-vacanaṃ ca icchāto vṛkṣā vṛkṣa iti syāt, yady eṣa niyamo bahuṣv eva bahu-vacanam, ekasminn eka-vacanam iti. asmākan tu sāṃketikeṣv artheṣu saṃketa- vaśād vṛttir ity anabhiniveśa eva.

[07,10]

nānā-eko rūpa-ādir eka-śabda-utthāpane śakta iti cet, kiṃ vai puruṣa-vṛtter anapekṣāḥ śabdān arthāḥ svayam utthāpayanti, āhosvit puruṣair śabdā vyavahāra-artham artheṣu niyujyante. svayam utthāpane hi bhāva-śaktir aśaktir vā cintyeta, na ca tad yuktam. puruṣais teṣāṃ niyoge yatheṣṭaṃ niyuñjīrann iti kas tatra upālambhaḥ, nimittaṃ ca niyogasya uktam eva. api ca yadi na rūpa-ādīnām ekena śabdena sambandhaḥ, katham ekena eṣām āśraya-abhimatena dravyeṇa sambandha iti kevalam ayam asad-bhūta-abhiniveśa eva.

[07,19]

na vayam eka-sambandha-virodhād ekaṃ śabdaṃ na icchāmaḥ, api tv abhinnānāṃ rūpa-ādīnāṃ ghaṭa-kambala-ādiṣu nānā- arthakriyā-śabda-virodhāt ta eka-rūpāḥ samudāya-antara-asambhāvinīm arthakriyām eva na kuryuḥ. tena tat-prakāśanāya ekaikena api śabdena na ucyeran. bhavatu nāma kasyacid ayaṃ vāñcchā bhaveyur eka-rūpā rūpa-ādayaḥ sarva-samudāyeṣv iti.

[07,25]

kim idaṃ paraspara-vivikta-rūpa pratibhāsa-adhiyakṣa-darśanam enām upekṣate. aniṣṭaṃ ca idaṃ rūpa-ādīnāṃ pratisamudāyaṃ svabhāva-bheda-upagamāt.

[08,01]

yady anya eva rūpa-ādibhyo ghaṭa iti ekaḥ syāt, kiṃ syāt. astu, pratyakṣasya sato arūpa-ādi-rūpasya tad-vivekena buddhau svarūpeṇa pratibhāsena kim āvaraṇam. pratibhāsamānāś ca vivekena pratyakṣā arthā dṛśyante apṛthagdeśatve api gandha-rasa-ādayaḥ, vāta-ātapa-sparśa-ādayaś ca eka-indriya-grāhyatve api. idam eva ca pratyakṣasya pratyakṣatvam, yad anātma-rūpa-vivekena svarūpasya buddhau samarpaṇam.

[08,08]

ayaṃ punar ghaṭo amūlya-dāna-krayī, yaḥ svarūpaṃ ca na upadarśayati pratyakṣatāṃ ca svīkartum icchati. etena buddhi- śabda-ādayo api vyākhyātā yadi tais tat-sādhanam iṣyate. na ca pratyakṣasya anabhibhave rūpa-anupalakṣaṇam, yena tat-sādhanāya liṅgam ucyate. apratyakṣatve apy apramāṇasya sattā- upagamo na yuktaḥ. tan na rūpa-ādibhyo anyo ghaṭaḥ. evaṃ tāvan na buddhi-vyapadeśābhyāṃ sattā-vyavahāraḥ sattā- bheda-abheda-vyavahāro vā ata eva na tad-viparyayād viparyayaḥ.

[08,17]

arthakriyātas tu sattā-vyavahāraḥ syāt, na sattā-bheda-abheda- vyavahāraḥ, ekasya apy aneka-arthakriyā-darśanāt, yathā pradīpasya vijñāna-varti-vikāra-jvālā-antara-utpādanāni, anekasya api cakṣur-āder eka-vijñāna-kriyā-darśanāt.

[08,21]

na brūmo arthakriyā-bheda-mātreṇa sattā-bheda iti, kiṃ tarhi adṛṣṭa-arthakriyā-bhedena. yā arthakriyā yasminn adṛṣṭā punar dṛśyate sā sattā-bhedaṃ sādhayati, yathā mṛdy adṛṣṭā saty udaka-dhāraṇa-ādy-arthakriyā ghaṭe dṛśyamānā, adṛṣṭā api tantuṣu prāvaraṇa-ādy-arthakriyā paṭe dṛśyata iti sattā- bhedaḥ.

[09,01]

sidhyati evam artha-antaram, tathā apy avayavī na sidhyati.

[09,02]

yathāpratyayaṃ saṃskāra-santatau svabhāva-bheda-utpatter arthakriyā-bhedaḥ, araṇi-nirmathana-avasthā-bhedād iva agneḥ sthūla-karīṣa-tṛṇa-kāṣṭha-dahana-śakti-bhedaḥ. tathā yathāpratyayaṃ svabhāva-bheda-utpattes tantv-ādiṣv arthakriyā-bhedaḥ.

[09,06]

etena buddhi-vyapadeśa-bheda-abhedau vyākhyātau.

[09,07]

tatra yad uktam arthakriyātaḥ sattā-vyavahāra-siddhiḥ, viparyayād viparyaya iti satyam etat. sa eva tu viparyayo anupalabdhi-lakṣaṇa-prāpteṣu na sidhyati. tatra punar idam anicchato apy āyātam, yasya idaṃ sāmarthyam upalabdhi-lakṣaṇa- prāptaṃ san na upalabhyate, so asad-vyavahāra-viṣayaḥ sāmarthya-lakṣaṇatvāt sattvasya iti. tathā api ko atiśayaḥ pūrvakād asya. na hi svabhāvād artha-antaraṃ sāmarthyam, tasya upalabdhi-lakṣaṇa-prāptasya yo anupalambhaḥ sa svabhāvasya eva iti pūrvakā eva iyam anupalabdhiḥ. tasmād anena kvacit keṣāṃcid asad-vyavahāram abhyupagacchatā ato anupalambhād abhyupagantavyo na vā kvacid viśeṣa-abhāvāt. so anyatra api tathāvidhe aviśiṣṭa iti so api tathā astv iti vyāptiḥ: sarva evaṃvidho anupalabdho asad-vyavahāra-viṣaya iti.

[09,20]

na eva kaścit kvacit kathaṃcid anupalabdho apy asad-vyavahāra- viṣaya iti cet, sarvasya sarva-rūpāṇāṃ sarvadā anivṛtteḥ sarvaṃ sarvatra sarvadā samupayujyeta. idaṃ ca na syāt: idam ataḥ, nāta idam, iha idam, iha na idam, idānīm idam, idānīṃ na idam, idam evam, idaṃ na evam iti, kasyacid api rūpasya kathaṃcid kvacit kadācid viveka-hetor abhāvāt.

[09,26]

ananvaya-vyatirekaṃ viśvaṃ syāt, bheda-abhāvāt.

[10,01]

avasthā-nivṛtti-pravṛtti-bhedebhyo vyavasthā iti cet, nanu ta eva sarva-viṣayasya asad-vyavahārasya abhāvān na sambhavanti, yatas tebhyo vyavasthā syāt. kvacid viṣaye asad-vyavahāra- upagame sa kuta iti vaktavyam. na hy anupalambhād anyo vyavaccheda-hetur asti, vidhi-pratiṣedhābhyāṃ vyavacchede sarvadā anupalambhasya eva sādhanatvāt. anupalambhād eva tad-abhyupagame sa yatra eva asti sarvo asad-vyavahāra- viṣaya iti vaktavyam, viśeṣa-abhāvāt.

[10,09]

sarva-pramāṇa-nivṛttir anupalabdhiḥ. sā yatra so asad-viṣaya iṣṭa iti cet, sukumāra-prajño devānāṃpriyo na sahate pramāṇa-cintā-vyavahāra-parikleśaṃ yena na atra ādaraṃ kṛtavān.

[10,12]

na hy anumāna-ādi-nivṛttir abhāvaṃ gamayati vyabhicārāt, na sarva-pratyakṣa-nivṛttir asiddheḥ, na ātma-pratyakṣa-aviśeṣa- nivṛttir api viprakṛṣṭeṣu. tasmāt svabhāva-viśeṣo yataḥ pramāṇān niyamena sad-vyavahāraṃ pratipadyate, tan-nivṛttis tasya asad-vyavahāraṃ sādhayati, tat-svabhāva-sattāyās tat- pramāṇa-sattayā vyāpteḥ. na ca upalabdhi-lakṣaṇa-prāptasya arthasya pratyakṣād anya-upalabdhir yena anumānād asya upalabdhiḥ syāt. na ca tad-rūpa-anyathābhāvam antareṇa apratyakṣatā- anyathābhāve ca tad eva na syāt.

[10,21]

api ca kuta idam amantra-oṣadham indrajālaṃ bhāvena śikṣitam, yad ayam ajāta-anaṣṭa-rūpa-atiśayo avyavadhāna-adūra- sthānas tasya eva tad-avasthā indriya-āder eva puruṣasya kadācit pratyakṣo apratyakṣaś ca, yena kadācit asya anumānam upalabdhiḥ kadācit pratyakṣaṃ kadācid āgamaḥ, etasminn eva anatiśaye amīṣāṃ prakārāṇāṃ virodhāt.

[11,05]

na anatiśaya eka-atiśaya-nivṛttyā apara-atiśaya-utpattyā ca vyavahāra- bheda-upagamāt. so atiśayas tasya ātmabhūto ananvayo nivartamānaḥ pravartamānaś ca kathaṃ na svabhāva-nānātvam ākarṣayati sukha-duḥkha-vat. sānvayatve ca kā kasya pravṛttir nivṛttir vā iti yat kiṃcid etat.

[11,10]

athavā yadi kasyacit svabhāvasya pravṛtti-nivṛttī svayam abhyanujñāyete, tad eva paro bruvāṇaḥ kim iti na anumanyate.

[11,12]

tasya niranvaya-upajanana-vināśa-upagamād iti cet, ko ayam anvayo nāma bhāvasya janma-vināśayoḥ. śaktiḥ, sā asty eva prāg api janmano nirodhād apy ūrdhvam. tena ayaṃ na apūrvaḥ sarvathā jāyate, na pūrvo vināśyati iti. yadi sā sarvadā anatiśayā kim idānīm atiśayavat yat kṛto ayaṃ vyavahāra-vibhāgaḥ.

[11,17]

tā avasthā atiśayavatya iti cet tā avasthāḥ sā ca śaktiḥ, kim eko bhāva āhosvin nānā. ekaś cet, katham idānīm idam ekatra avibhakta-ātmani niṣparyāyaṃ paraspara-vyāhataṃ yokṣyate: janma-ajanma nivṛttir anivṛttir ekatvaṃ nānātvaṃ pratyakṣatā-apratyakṣatā arthakriyā-upayogo anupayogaś ca ity ādi.

[12,01]

asti paryāyo avasthā śaktir iti tena avirodha iti cet, vismaraṇa- śīlo devānāṃpriyaḥ prakaraṇaṃ na lakṣayati. śaktir avasthā ity eko bhāvo avibhāga iti pakṣe ayaṃ virodha uktaḥ.

[12,04]

atha asty anayor vibhāgo na kaścid virodhaḥ, kevalaṃ sānvayau bhāvasya janma-vināśāv iti na syāt, yasmād yasya anvayo na tasya janma-vināśau yasya ca tau na tasya anvayaḥ.

[12,07]

tayor abhedād adoṣa iti cet, anuttaraṃ bata, doṣa-saṅkaṭam atrabhavān dṛṣṭi-rāgeṇa praveśyamāno api na ātmānaṃ cetayati. abhedo hi nāma-aikyaṃ tāv iti ca bheda-adhiṣṭhāno bhāviko vyavahāraḥ. nivṛtti-prādurbhāvayor anivṛtti-prādurbhāvau sthitāv, asthitir ity ādikaṃ nānātva-lakṣaṇaṃ ca kathaṃ yojyate. eṣa hi bhāvānāṃ bheda etad-virahaś ca abhedo yathā sukha-ādiṣu śakty- avasthayoś ca ekātmani. anyathā bheda-abheda-lakṣaṇa-abhāvād bheda-abhedayor avyavasthā syāt sarvatra.

[12,16]

tadātmani prādurbhāvo abhedaḥ, viparyaye bhedaḥ, yathā mṛd-ātmani prādurbhavato ghaṭasya tasmād abhedaḥ.

[12,18]

bhedaś ca viparyaye sukha-duḥkhayor iti, idaṃ bheda- abheda-lakṣaṇam, tena avirodha iti cet, na vai mṛdātmani ghaṭasya prādurbhāvaḥ, kiṃ tarhi mṛdātmā eva kaścit ghaṭaḥ. na hy ekas trailokye mṛd-ātmā, prativijñapti-pratibhāsa- bhedāt dravya-svabhāva-bhedāt. evaṃ hy asya api sukha-ādiṣu caitanyeṣu ca bheda-avagamaḥ samartho bhavati, yady evaṃ bhedaḥ syāt.

[13,01]

saty apy etasmin kasyacid ātmano anvayād aikyam iti cet, sukha-ādiṣv apy ayaṃ prasaṅgaḥ caitanyeṣu ca. na ca ghaṭa- ādiṣv api sarva-ātmanā anvayo avaiśvarūpya-sahotpatty-ādi-prasaṅgāt.

[13,04]

na ca ghaṭaṃ mṛd-ātmānaṃ ca kaścid vivekena upalakṣayati.

[13,05]

yena evaṃ syād idam iha prādurbhūtam iti. na hy adhiṣṭhāna-adhiṣṭhāninor vivekena anupalakṣaṇe evaṃ bhavati.

[13,07]

na ca śakteḥ śakty-ātmani prādurbhāva iti tasyāḥ svātmano abhedo na syāt.

[13,09]

etena pariṇāmaḥ pratyuktaḥ, yo api hi kalpayet, yo yasya pariṇāmaḥ, sa tasmād abhinna iti, na hi śakter ātma-apariṇāma iti, kiṃ ca idam uktaṃ bhavati pariṇāma iti. avasthitasya dravyasya dharma-antara-nivṛttir dharma-antara-prādurbhāvaś ca pariṇāmaḥ yat tad dharma-antaraṃ nivartate prādurbhavati ca, kiṃ tat tad eva avasthitaṃ dravyaṃ syāt.

[13,15]

tato artha-antaraṃ vā anya-vikalpa-abhāvāt.

[13,16]

yadi tat tad eva, tasya avasthānaṃ na nivṛtti-prādurbhāvāv iti kasya tāv iti vaktavyam. avasthitasya dharma-antaram iti ca na sidhyati. na hi tad eva tasya anapāśrita-vyapekṣa-abhedaṃ dharma-antaraṃ bhavati.

[13,20]

atha dravyād artha-antaraṃ dharmaḥ, tadā tasya nivṛtti- prādurbhāvābhyāṃ na dravyasya pariṇatiḥ. na hy artha-antaragatābhyāṃ nivṛtti-prādurbhāvābhyām artha-antarasya pariṇatiḥ, caitanye api prasaṅgāt.

[14,01]

dravyasya dharma iti ca vyapadeśo na sidhyati sambandha- abhāvāt. na hi kāryakāraṇabhāvād anyo vastu-sambandho asti. na ca anayoḥ kāryakāraṇabhāvaḥ, svayam atadātmano atat-kāraṇatvāt, dharmasya dravyād artha-antara-bhūtatvāt.

[14,06]

arthāntaratve api dharma-kāraṇatve dharma-antarasya kāryasya utpādanād dravyasya pariṇāma iti iṣṭaṃ syāt. tad aviruddham anyasya api, hetu-phala-santāne mṛd-dravya-ākhye pūrvakān mṛt-piṇḍa-dravyāt kāraṇād uttarasya ghaṭa-dravyasya kāryasya utpattau mṛd-dravyaṃ pariṇatam iti vyavahārasya upagamāt.

[14,11]

na ca dharmasya dravyāt tattvānyatvābhyām anyo vikalpaḥ sambhavati, ubhayathā api na pariṇāmaḥ.

[14,13]

na nirvivekaṃ dravyam eva dharmaḥ, na api dravyād artha-antaram, kiṃ tarhi, dravyasya saṃniveśo avasthā-antaram, yathā aṅgulīnāṃ muṣṭiḥ. na hy aṅguly eva nirvivekā muṣṭiḥ, prasāritānām amuṣṭitvāt, na apy artha-antaram, pṛthak-svabhāvena anupalabdher iti cet, na, muṣṭer aṅguli-viśeṣatvāt. aṅgulya eva hi kāścin muṣṭiḥ, na sarvāḥ. na hi prasāritā aṅgulyo nirviveka-svabhāvā muṣṭy-aṅgulyaḥ, avasthā-dvaye api ubhaya- pratipatti-prasaṅgāt. ya eva hi khalu vivekaḥ svabhāvabhūtaḥ, sa eva vastu-bheda-lakṣaṇaṃ sukha-duḥkha-vat. parabhūte ca viveka-utpāde aṅgulyaḥ prasāritā eva upalabhyeran, na hi svayaṃ svabhāvād acyutasya artha-antara-utpāde anyathā-upalabdhiḥ, atiprasaṅgāt nanu uktaṃ na dravyam eva nirvivekam avasthā, na api dravyād artha-antaram iti. uktam idam, na punar yuktam, na hi sato vastunas tattvānyatve muktvā anyaḥ prakāraḥ sambhavanti, tayor vastuni paraspara-parihāra-sthita-lakṣaṇatvena eka-tyāgasya apara-upādāna-nāntarīyakatvāt. aṅgulīṣu punaḥ pratikṣaṇa-vināśinīṣv anyā eva prasāritāḥ, anyā muṣṭiḥ.

[15,10]

tatra muṣṭy-ādi-śabdā viśeṣa-viṣayāḥ, aṅgulī-śabdaḥ sāmānya- viṣayaḥ, bīja-aṅkura-ādi-śabda-vat vrīhy-ādi-śabda-vac ca. tena aṅgulyaḥ prasāritā na muṣṭiḥ.

[15,13]

tad yadi prāg asad eva kāraṇe kāryaṃ bhavet, kiṃ na sarvaḥ sarvasmād bhavati. na hy asattve kaścid viśeṣa iti.

[15,15]

nanu sarvatra sarvasya sattve apy ayaṃ tulyo doṣaḥ. na hi sattve kaścid viśeṣaḥ. viśeṣe vā sa viśeṣas traiguṇyād bhinnaḥ syāt, tadbhāve viśeṣasya ananvayāt. sataś ca sarvātmanā niṣpanna-avasthāyām iva kiṃ jāyate. sādhana-vaiphalyaṃ ca, sādhyasya kasyacid abhāvāt yasya kasyacid atiśayasya tatra kathaṃcid asata utpattau, so atiśayas tatra asan kathaṃ jāyate. jātau vā sarvaḥ sarvasmāj jāyeta iti tulyaḥ paryanuyogaḥ.

[16,01]

na atiśayas tatra sarvathā na asti, kathaṃcit sata eva bhāvād iti cet, yathā na asti, sa prakāras tatra asan kathaṃ jāyeta.

[16,03]

na ca sarvathā sataḥ kaścij janma-artha ity uktam.

[16,04]

asato api kāryasya kāraṇād utpāde, yo yaj-janana-svabhāvaḥ, tata eva tasya janma, na anyasmād iti niyamaḥ.

[16,06]

tasya api sa svabhāva-niyamaḥ sva-hetor ity anādi-bhāva-svabhāva- niyamaḥ.

[16,08]

api ca, yadi mṛt-piṇḍe ghaṭo asti, kathaṃ tad-avasthāyāṃ na paścād-vad upalabdhiḥ, tad-arthakriyā vā. vyakter aprādurbhāvād iti cet, tasya eva tad-arthakriyā-ādi-bhāve ghaṭatvāt, tad-rūpasya ca prāg asattvāt, kathaṃ ghaṭo asti. na hi rūpa-antarasya sattve rūpa-antaram asti. na ca rūpa-pratibhāsa-bhede vastv- abhedo yuktaḥ, atiprasaṅgāt.

[16,14]

tasmād ya upalabdhi-lakṣaṇa-prāpta-svabhāvo anupalabdhaḥ, sa na asty eva. na hi tasya tat-svabhāva-sthitāv anupalabdhiḥ.

[16,16]

tad-asthitiś ca atattvaṃ. paraspara-svabhāva-asthitayor iva duḥkha-sukhayor iti vyāptir asad-vyavahāra-niścayena anupalabdhi-viśeṣasya.

{111.121c3}

[16,19]

tena anupalabdhyā kasyacid vyavacchedaṃ prasādhayatā tasya yathokta-upalabdhi-lakṣaṇa-prāptir upadarśaniyā. upadarśya- anupalabdhi-nirdeśaḥ samarthanaṃ svabhāva-anupalabdhau.

[16,22]

vyāpaka-anupalabdhāv api dharmayor vyāpyavyāpakabhāvaṃ prasādhya vyāpakasya nivṛtti-prasādhanaṃ samarthanam.

[16,24]

kāraṇa-anupalabdhāv api kāryakāraṇabhāvaṃ prasādhya kāraṇasya nivṛtti-prasādhanaṃ samarthanam. tad- viruddha-upalabdhiṣv api dvayor viruddhayor ekasya viruddhasya upadarśanaṃ samarthanam.

[17,01]

evam anupalabdhau sādhana-aṅgasya asamarthanaṃ sādhana- aṅga-avacanaṃ tad vādino nigrahasthānam, asamarthane tasmin sādhya-asiddheḥ.

{111.2}

[17,04]

athavā sādhyate tena pareṣām apratīto artha iti sādhanaṃ trirūpa-hetu-vacana-samudāyaḥ. tasya aṅgaṃ pakṣa-dharma- ādi-vacanam. tasya ekasya apy avacanam asādhana-aṅga-vacanam.

[17,07]

tad api vādino nigrahasthānam tad-avacane hetu-rūpasya eva avacanena siddher abhāvāt.

{112.1}

[17,09]

athavā tasya eva sādhanasya yan na aṅgaṃ pratijñā-upanaya- nigamana-ādi, tasya asādhana-aṅgasya sādhana-vākye upādānaṃ vādino nigrahasthānam, vyartha-abhidhānāt.

[17,12]

anvaya-vyatirekayor vā sādharmyavati vaidharmyavati ca sādhana-prayoga ekasya eva abhidhānena siddher bhāvāt, dvitīyasya asāmarthyam iti tasya apy asādhana-aṅgasya abhidhānaṃ nigrahasthānam vyartha-abhidhānād eva.

[17,16]

nanu ca viṣaya-upadarśanāya pratijñā-vacanam asādhana-aṅgam apy upadeyam eva. na, vaiyarthyāt. asaty api pratijñā- vacane yathoktāt sādhana-vākyād bhavaty eva iṣṭa-artha-siddhir ity apārthakaṃ tasya upādānam. yadi ca viṣaya-upadarśanam antareṇa pratīter anutpattiḥ, kathaṃ na pratijñā sādhana- avayavaḥ. na hi pakṣa-dharma-ādi-vacanasya api pratīti-hetu- bhāvād anyaḥ sādhana-arthaḥ. sa pratijñā-vacane api tulya iti kathaṃ na sādhanam.

[18,01]

kevalasya asāmarthyād asādhanatvam iti cet, tat tulyaṃ pakṣa-dharma-vacanasya api iti tad api na sādhana-avayavaḥ syāt.

[18,03]

na hi pakṣa-dharma-vacanāt kevalāt pratipatter utpattiḥ.

[18,04]

etena saṃśaya-utpattiḥ pratyuktā, pakṣa-dharma-vacanād api kevalād apradarśite sambandhe saṃśaya-utpatteḥ. tasmād vyartham eva sādhana-vākye pratijñā-vacana-upādānaṃ vādino nigrahasthānam.

[18,08]

athavā sādhanasya siddher yan na aṅgam asiddho viruddho anaikāntiko vā hetvābhāsaḥ. tasya api vacanaṃ vādino nigrahasthānam asamartha-upādānāt.

[18,11]

tathā sādhya-ādi-vikalasya ananvaya-apradarśita-anvaya-āder api dṛṣṭāntābhāsasya asādhana-aṅgasya vacanam api vādino nigrahasthānam, asamartha-upādānād eva. na hi tair hetoḥ sambandhaḥ śakyate pradarśayitum, apradarśanād asāmarthyam.

{112.2_113.3}

[18,16]

athavā siddhiḥ sādhanam tad-aṅgaṃ dharmo yasya arthasya vivāda-āśrayasya vāda-prastāva-hetoḥ, sa sādhana-aṅgaḥ.

[18,18]

tad-vyatirekeṇa aparasya apy ajijñāsitasya viśeṣasya śāstra-āśraya- vyāja-ādibhiḥ prakṣepo ghoṣaṇaṃ ca para-vyāmohana-anubhāṣaṇa- śakti-vighāta-ādi-hetoḥ. tad apy asādhana-aṅga-vacanaṃ vādino nigrahasthānam, aprastuta-abhidhānāt, tathā viśeṣa-sahitasya arthasya prativādino ajijñāsitatvāt. jijñāsāyām adoṣaḥ.

[19,03]

jijñāsitaṃ punar arthasya anyasya prasaṅga-paraṃparayā yeṣa panna-ādinā bahiḥ prativādinaḥ prāśnikānāṃ ca nyāyadarśinām iti. ebhiḥ kathā-viccheda eva karaṇīyaḥ, na hi kaścid arthaḥ kvacit kriyamāṇa-prasaṅge na prayujyate, nairātmyavādinas tu tat-sādhane nṛtya-gīty-āder api tatra prasaṅgāt.

[19,09]

yathā pratijñā-abhidhāna-pūrvakaṃ kaścit kuryāt. na asty ātmā iti vayaṃ bauddhā brūmaḥ. ke bauddhāḥ. ye buddhasya bhagavataḥ śāsanam abhyupagatāḥ. ko buddho bhagavān.

[19,12]

yasya śāsane bhadanta-aśvaghoṣaḥ pravrajitaḥ. kaḥ punar bhadanta-aśvaghoṣaḥ. yasya rāṣṭra-pālaṃ nāma nāṭakam.

[19,14]

kīdṛśaṃ rāṣṭra-pālaṃ nāma nāṭakam iti prasaṅgaṃ kṛtvā nāndyante tataḥ praviśati sūtradhāra iti paṭhen nṛtyed gāyec ca. prativādī taṃ ca sarva-prasaṅgaṃ na anukartuṃ samartha iti parājitaḥ syād iti.

[19,18]

sabhyaḥ sādhu-sammatānāṃ viduṣāṃ tattva-cintā-prakāraḥ.

[19,19]

na ca evaṃ prastutasya paryavasānaṃ sambhavati, aniścaya- phalatvād anārambha eva vādasya.

[20,01]

kathaṃ ca evaṃ jaya-parājayau, prativādino apy ananubhāṣaṇasya evaṃprakārasya prasaṅgasya vistareṇa anubhāṣaṇa- vyājena sambhavād aniścitatvāc ca. tasmāt pratijñā-vacanam eva tāvan na nyāyyam. kutaḥ punar tatra ajijñāsita- viśeṣa-prasaṅga-upanyāsas tad-vyākhyā-prasaṅga-vitatha-pralāpaś ca. sarvaś ca ayaṃ prakāro durmatibhiḥ śaṭhair nyāya-sāmarthyena artha-pratipādane asamarthaiḥ pravartitaḥ. yathā puruṣa-atiśaya-pūrvakāni tanu-karaṇa-bhuvana-ādīni iti pratijñāya tanu-karaṇa-bhuvana-vyākhyā-vyājena sakala-vaiśeṣika-śāstra-artha- ghoṣaṇam, nityaḥ śabdo anityo vā iti vāde dvādaśa-lakṣaṇa- prapañca-prakāśana-śāstra-praṇetur jaimini-pratijñata-tattva- nityatā-adhikaraṇa-śabda-ghaṭa-anyatara-sadvitīyo ghaṭa iti pratijñām uparacayya dvādaśa-lakṣaṇa-artha-vyākhyānam. sarvo ayaṃ durmatīnām asāmarthya-pracchādana-upāyaḥ, na tu satyair asty upetaḥ, tattva-parīkṣāyāṃ phala-ādi-pratisaraṇa- daṇḍa-prayoga-ādīnām ayuktatvāt.

[20,17]

bhavaty eva nāṭaka-ādi-ghoṣaṇe artha-antara-gamanāt parājaya iti cet, anyasya apy ajijñāsitasya kiṃ na bhavati. na hi asya api kācid vivakṣita-sādhya-dharma-siddhau nāntarīyakatā.

[20,20]

yathā hetu-pratyaya-pāratantrya-lakṣaṇa-saṃskāra-duḥkhatā- ādi-siddhim antareṇa na anityatā-siddhiḥ, tathāvidhas tu dharmaḥ pṛthag anukto api sādhya-dharme antarbhāvāt pakṣīkṛta eva iti na pṛthag asya upanyāso vyākhyānaṃ vā. tasmād evaṃvidhasya api tadānīṃ prativādinā ajijñāsitasya arthasya pratijñāyām anyatra vā upanyāso vyākhyānaṃ vā artha-antara- gamanān nigrahasthānam eva. tena jijñāsita-dharma-mātram eva sādhana-aṅgaṃ vācyam. na prasaṅga upakṣeptavyaḥ, tad-upakṣepe atiprasaṅgāt.

[21,06]

evam asādhana-aṅga-vacanaṃ vādino nigrahasthānaṃ prativādinā tathābhāve pratipādite, anyathā dvayor ekasya api na jaya-parājayāv iti.

[21,09]

adoṣa-udbhāvanaṃ prativādino nigrahasthānam. vādinā sādhane prayukte abhyupagata-uttara-pakṣo yatra viṣaye prativādī yadā na doṣam udbhāvayati tadā parājito vaktavyaḥ.

[21,12]

sādhana-doṣāḥ punar nyūnatvam asiddhir anaikāntikatā vādinaḥ sādhayitum iṣṭasya arthasya viparyaya-sādhanam aṣṭādaśa dṛṣṭānta-doṣaś ca. teṣām anudbhāvanam apratyāyanam apratipādanaṃ prativādinaḥ parājaya-adhikaraṇam.

[21,16]

tat punaḥ sādhanasya nirdoṣatvāt, sadoṣatve api prativādino ajñānāt pratipādana-asāmarthyād vā. na hi duṣṭa-sādhana-abhidhāne api vādinaḥ prativādino apratipādite doṣe parājaya- vyavasthāpanā yuktā, tayor eva paraspara-sāmarthya-upaghāta- apekṣayā jaya-parājaya-vyavasthāpanāt. kevalaṃ hetvābhāsād bhūta-pratipatter abhāvād apratipādakasya jayo api na asty eva. na hi tattva-cintāyāṃ kaścic chala-vyavahāraḥ.

[22,01]

yady evaṃ kiṃ na parājayaḥ, tattva-siddhi-bhraṃśāt.

[22,02]

na anirākaraṇāt. nirākaraṇaṃ hi tasya anyena parājayaḥ, na siddhy-abhāvaḥ, pratiyogy-anapekṣaṇāt siddhy-abhāvasya, sādhana-abhāve asaty api pratiyogini bhāvāt. pratiyoginaś ca tan-nirākaraṇe asāmarthyāt parājayasya anutpatter aparājayaḥ.

[22,06]

tasmād ayam asamartha-sādhana-abhidhāyy api pareṇa tathābhāve apratipādite aparājito vaktavyaḥ.

[22,08]

chala-vyavahāre api vijigīṣūṇāṃ vāda iti cet, na, durjana- vipratipatty-adhikāre satāṃ śāstra-pravṛtteḥ. na hi para-anugraha- pravṛttā mithyā-pralāpa-ārambha-ātma-utkarṣa-para-paṃsana- ādīn asad-vyavahārān upadiśanti. na ca para-vipaṃsanena lābha-satkāra-śloka-upārjanaṃ satām ācāraḥ. na api tathā-pravṛttebhyaḥ sva-hasta-dānena prāṇinām upatāpanaṃ sat-sammatānāṃ śāstrakāra-sabhāsadāṃ yuktam. na ca nyāya-śāstrāṇi sadbhir lābha-ādy-upārjanāya praṇīyante. tasmān na yoga-vihitaḥ kaścid vijigīṣu-vādo nāma. para-anugraha-pravṛttās tu santo vipratipannaṃ pratipādayanto nyāyam anusareyuḥ sat- sādhana-abhidhānena bhūta-doṣa-udbhāvanena vā, sākṣi-pratyakṣaṃ tasya eva anuprabodhāya. tad eva nyāya-anusaraṇaṃ satāṃ vādaḥ, ukta-nyāye tattva-arthī cet pratipadyeta, tad-apratipattāv apy anyo na pratipadyeta iti.

[22,22]

tattva-rakṣaṇa-arthaṃ sadbhir upahartavyam eva chala-ādi vijigīṣubhir iti cet, na, nakha-capeṭa-śastra-prahāra-ādīpana-ādibhir api iti vaktavyam. tasmān na jyāyān ayaṃ tattva-rakṣaṇa- upāyaḥ. sādhana-prakhyāpanaṃ satāṃ tattva-rakṣaṇa-upāyaḥ sādhana-ābhāsa-dūṣaṇaṃ ca, tad-abhāve mithyā-pralāpād atra para-upatāpa-vidhāne api tattva-apratiṣṭhāpanāt. anyathā api nyāya- upavarṇane vidvat-pratiṣṭhānāt. tasmāt para-anugrahāya tattva-khyāpanaṃ vādino vijayaḥ, bhūta-doṣa-darśanena mithyā- pratipatti-nivartanaṃ prativādinaḥ.

{122+}

[23,07]

athavā yo na doṣaḥ sādhanasya tad-bhāve api vādinā sādhayitum iṣṭasya arthasya siddher vighāta-abhāvāt, tasya udbhāvanaṃ prativādino nigraha-adhikaraṇam, mithyā-uttara-abhidhānāt.

[23,11]

yathā sādhyatayā aniṣṭo api vādino dharmaḥ, śāstra-upagamāt sādhya iti tad-viparyāsena virodha-udbhāvanam. na asty ātmā iti tava pratijñā-padayor virodha iti pratijñā-doṣa-udbhāvanam.

[23,14]

prayatnānantarīyakaḥ śabdo anityaḥ prayatnānantarīyakatvād iti hetor dharma-viśeṣatvāt pratijñārthaikadeśa ity asiddha-udbhāvanam, sarvāṇi sādharmya-vaidharmya-sama-ādīni jāty-uttarāṇi ity evam-āder doṣasya udbhāvanam adoṣa-udbhāvanam.

[23,19]

tasya vādinā doṣa-ābhāsatve prakhyāpite prativādī parājito vaktavyaḥ, pūrvapakṣe sādhanasya nirdoṣatvāt. doṣavati punaḥ sādhane na dvayor ekasya api jaya-parājayau, tattva- aprakhyāpanāt adoṣa-udbhāvanāc ca. apratipakṣāyāṃ ca pakṣa-siddhau kṛtāyāṃ jetā bhavati.

[24,01]

tasmāj jigīṣatā sva-pakṣaś ca sthāpanīyaḥ para-pakṣaś ca nirākartavyaḥ. nirdoṣe sādhana-abhidhāne api vādinaḥ prativādinā doṣa-ābhāsa udbhāvite, dūṣaṇa-ābhāsatva-khyāpana eva jaya-parājayau, na anyathā, bhāvatas tattva-abhidhāne api pratipakṣa- nirākaraṇena tattvasya prakhyāpana-asāmarthyāt. na prativādino apy atra, bhāvato mithyā-pratipatter iti.

[24,07]

idaṃ nyāyyaṃ nigrahasthāna-lakṣaṇam uktam asmābhiḥ.

{2.0}

<nyāyamatakhaṃḍana>

[25,01]

anyat tu na yuktam iti na iṣyate.

[25,02]

yatra idaṃ yathoktaṃ nigrahasthāna-lakṣaṇaṃ na asti tasya nigrahasthānatvam ayuktam iti na uktam asmābhiḥ.

{2.1}

[25,04]

pratidṛṣṭānta-dharma-abhyanujñā-sva-dṛṣṭānte pratijñāhānir nigrahasthānam iti atra bhāṣyakāra-mataṃ dūṣayitvā vārttikakāro ayaṃ sthita-pakṣam āha. tam eva brūmaḥ.

[25,07]

pratidṛṣṭāntasya yo dharmas taṃ yadā sva-dṛṣṭānte abhyanujānāti nigṛhīto veditavyaḥ. tatra dṛṣṭaś ca asāv ante ca vyavasthita iti dṛṣṭāntaḥ. sva-dṛṣṭāntaḥ sva-pakṣaḥ, pratidṛṣṭāntaḥ pratipakṣaḥ. pratipakṣasya dharmaṃ sva-pakṣe abhyanujānan parājitaḥ, yathā anityaḥ śabda aindriyakatvād iti brūvan pratipakṣa-vādini sāmānyena pratyavasthite āha yadi sāmānyam aindriyakaṃ nityaṃ śabdo apy evam astv iti eṣā pratijñāhāniḥ prāk-pratijñātasya śabda-anityatvasya tyāgād iti.

[25,16]

atra upagata-pratijñā-tyāgāt pratijñāhānau viśeṣa-pratiniyamaḥ kiṃ kṛto anena prakāreṇa pratijñāṃ tyajataḥ pratijñāhānir iti. sambhavati hy anyena api prakāreṇa hetu-doṣa-udbhāvana- ādinā pratipakṣa-sādhana-abhidhānena ca sva-pakṣa-parityāgaḥ para-pakṣa-upagamaś ca. idam eva ca pratijñāhāneḥ pradhānaṃ nimittam evaṃ pratipāditena pratijñā hātavyā hānau ca parājaya iti.

[26,01]

idaṃ punar asambaddham eva sāmānyaṃ nityam aindriyakam ity ukte śabdo apy evam astv iti. kaḥ svastha-ātmā svayam aindriyakatvād anityaḥ śabdo ghaṭavad iti bruvan sāmānyena upadarśana-mātreṇa nityaṃ śabdaṃ pratipadyate.

[26,05]

sāmānyasya api nityasya aindriyakatve apy anitye ghaṭe darśanāt saṃśayitaḥ syāt.

[26,07]

jāḍyāt pratipadyeta api iti cet, tathā api kiṃ sāmānyasya upadarśanena.

[26,08]

evam eva nityaḥ śabda iti vaktavyaṃ jaḍasya pratipattau vicāra-abhāvāt.

[26,10]

na ca nitya-sāmānya-upadarśanena tad-dharmaṃ śabde pratipadyamāne pratipakṣa-dharma-abhyanujñāto bhavati, anityaḥ śabda iti ca vadato nityaḥ śabda ity āñjasaḥ pratipakṣaḥ syāt, na nityaṃ sāmānyam iti.

[26,14]

tasmād aindriyakatvasya nitya-anitya-pakṣa-vṛtter vyabhicārād asādhana-aṅgasya upādānān nigraha-arhaḥ, na pratipakṣa- dharma-anujñayā anena prakāreṇa pratijñāhāneḥ.

{2.2}

[26,17]

pratijñā-artha-pratiṣedhe dharma-vikalpāt tad-artha-nirdeśaḥ pratijñā-antaram pratijñāto artho anityaḥ śabda aindriyakatvād ity eva, tasya hetu-vyabhicāra-upadarśanena pratiṣedhe kṛte dharma-bheda-vikalpāt sāmānya-ghaṭayoḥ sarvagatatva- asarvagatatva-dharma-vikalpena pratijñā-antaraṃ karoti, yathā ghaṭo asarvagato anitya eva śabdo apy asarvagato anitya iti. etat pratijñā-antaraṃ nāma nigrahasthānaṃ sādhana-sāmarthye apy aparijñānāt. sa hi pūrvasyā anityaḥ śabda iti pratijñāyāḥ sādhanāya uttarām asarvagataḥ śabda iti pratijñām āha. tad-darśanāya tad-artha-nirdeśa ity āha. tad-arthaḥ pūrva-ukta-sādhya-siddhy-artha uttara-pratijñā-nirdeśas tad-artha- nirdeśaḥ. na ca pratijñā pratijñā-antara-sādhane samarthā iti nigrahasthānam.

[27,10]

atra api na evam-bruvatā pratijñā-antaraṃ pūrva-pratijñā-sādhanāya uktaṃ bhavati, kiṃ tarhi viśeṣaṇam aindriyakatvasya hetoḥ sāmānye vṛttyā vyabhicāra udbhāvite asarvagatatve saty aindriyakatvasya hetor viśeṣaṇa-upādāne vyabhicāraṃ pariharati. na punaḥ pratijñā-antaram āha, asarvagatatvasya śabde siddhatvāt pratijñāyāś ca sādhya-nirdeśa-lakṣaṇatvāt.

[27,17]

yad apy uktaṃ pūrva-pratijñā-sādhanāya uttarāṃ pratijñām āha iti tad apy ayuktam. na hi pratijñā pratijñā-sādhanāya ucyamānā pratijñā-antaraṃ bhavati, kiṃ tarhi hetv-āder ananyatamaḥ.

[27,20]

sādhya-sādhanāya upādānāt sādhana-nirdeśaḥ sa syān na sādhya-nirdeśaḥ. udāharaṇa-sādharmya-ādeś ca hetu-lakṣaṇasya asarvagatve bhāvāt pratijñā-lakṣaṇasya ca abhāvāt, hetutvam asarvagatve prayuktaṃ na pratijñā-antaratvam. atyanta- asambaddhaṃ ca idaṃ pratijñāṃ pratijñā-sādhanāya āha iti.

[28,05]

yo hi prāk pratijñām uktvā hetu-udāharaṇa-ādikaṃ vaktuṃ jānāti, sa kiṃcid anukramaṃ sādhanasya jānāty eva hi, jānan katham avikala-antaḥkaraṇaḥ pratijñām eva pratijñā-sādhanāya upādadīta. upādadatā ca anena pratijñā-mātreṇa siddhir iṣṭā bhavati, tataś ca na prāg api hetuṃ brūyāt.

[28,10]

evaṃprakāraṇām asambaddhānāṃ parisaṃkhyātum aśakyatvāt, lakṣaṇa-niyamo apy asambaddha eva pratijñā- antara-abhidhāne pratijñā-antaraṃ nāma nigrahasthānam iti.

[28,13]

asambaddha-abhidhānaṃ nigrahasthānam iti evaṃprakārāṇām ekam eva lakṣaṇaṃ vācyaṃ syāt. na ca evaṃvidhaḥ kaścid vivādeṣu dṛṣṭa-pūrvo vyavahāro yena tad-arthaṃ yatnaḥ kriyate. na ca bāla-pralāpa-anudiśya śāstraṃ pravartate pravṛttau ca kā niṣṭhā teṣām aniṣṭānāt. dṛśyate ca viduṣām api na atinirūpaṇād asiddha-abhidhānam iti. vyavahāra- darśanāt tādṛśaṃ parājaya-adhikaraṇaṃ vyavasthāpyate.

[29,01]

tasmād iha api yadi nivṛtta-ākāṅkṣe vādini paro anaikāntikatām udbhāvayet, asādhana-aṅgasya anaikāntikasya abhidhānān nigrahasthānaṃ vādinaḥ, evaṃ yadi prativādī sat sāmānyam aindriyakaṃ nityaṃ ca pramāṇena pratipādayituṃ śaknuyāt. anudiśya-apramāṇakaṃ śāstra-upagamaṃ pramāṇena eṣām arthānām apratipādane na bhūta-doṣa-udbhāvanam iti na kaścit parājayo abhyupagama-mātreṇa vastu-siddher abhāvāt prativādinā doṣasya apratipāditatvāt. pramāṇair asamarthita- sādhana-abhidhānāt tu jetā api na bhavati iti.

[29,10]

anitya-ākāṅkṣe punar vādini na kaścid doṣo viśeṣaṇa-abhidhānena hetoḥ samarthana-upakramāt.

{2.3}

[29,12]

pratijñā-hetvor virodhaḥ pratijñā-virodhaḥ. yathā guṇa-vyatiriktaṃ dravyam iti pratijñā, rūpa-ādibhyo artha-antarasya anupalabdher iti hetuḥ, so ayaṃ pratijñā-hetvor virodhaḥ. etena eva pratijñā-virodho apy ukto yatra pratijñā sva-vacanena virudhyate, yathā śramaṇā garbhiṇī, na asty ātmā iti vā, hetu- virodho api yatra pratijñayā hetur virudhyate, yathā sarvaṃ pṛthak samūhe bhāva-śabda-prayogād iti. etena pratijñayā dṛṣṭānta-virodho api vyākhyātaḥ. hetoś ca dṛṣṭānta-ādibhir virodhaḥ pramāṇa-virodhaś ca pratijñā-hetvor vaktavyaḥ.

[30,01]

para-pakṣe sva-siddhena gotva-ādinā anaikāntika-codanād virodhaḥ, yaḥ para-pakṣaṃ sva-siddhena gotva-ādinā vyabhicārayati, tad viruddham uttaraṃ veditavyam sva-pakṣa-anapekṣaṃ ca, yaś ca sva-pakṣa-anapekṣaṃ hetuṃ prayuṅkte anityaḥ śabda aindriyakatvād iti, tasya sva-siddhasya gotva-āder anityatva-virodhād viruddhaḥ. ubhaya-pakṣa-sampratipannas tv anaikāntikaḥ, yad ubhaya-pakṣa-pratipannaṃ vastu, tena anaikāntika-codanā iti.

[30,09]

atra api pratijñāyāḥ sādhana-vākye prayoga-pratiṣedhāt tad-āśrayas tat-kṛto vā hetu-dṛṣṭāntayor na virodha iti na pratijñā- virodho nāma kiṃcit nigrahasthānam.

[30,12]

syād etat, asaty api pratijñā-prayoge gamyamāno api pratijñā- hetvor virodho bhavati, yathā rūpa-ādibhyo artha-antarasya anupalabdhiḥ tad guṇa-vyatiriktam, na upalabhyate ca rūpa-ādibhyo artha-antaraṃ dravyam ity ukte api gamyata eva sādhya- sādhanayor virodhaḥ; kathaṃ tato artha-antarasya-anupalabdhis tad-vyatirekaś ca iti.

[30,18]

satyam, syād ayaṃ virodhaḥ, yadi hetuḥ sādhya-dharma- viparyayaṃ sādhayet. yadi hy upalabdhi-lakṣaṇa-prāptatvena upagatasya sato dravyasya rūpa-ādi-pratibhāsa-vivekena sva- pratibhāsa-anupalabdhiḥ, tat tad-vyatirekeṇa na asti iti iṣṭa-vyatireka- viparyaya-sādhanād viruddho hetur asmābhir ukta eva iti bhavaty eva idaṃ nigraha-adhikaraṇam, yady evaṃvidhaḥ pratijñā- hetvor virodha iṣṭaḥ. atha punar asya upalabdhi-lakṣaṇa- prāptir lupyate, tadā na kaścit pratijñā-hetvor virodhaḥ, vyatiriktānām api bhāvānāṃ kutaścid viprakarṣiṇāṃ tad- vyatirekeṇa anupalabdhāv api vyatirekasya bhāvāt.

[31,10]

yad uktaṃ pratijñāyāḥ sva-vacana-virodhe pratijñā-virodha iti tatra idam eva nigraha-adhikaraṇam asādhana-aṅga- bhūtāyāḥ pratijñāyāḥ sādhana-vākye prayogaḥ; na virodhaḥ tad-adhikaraṇatvāt. yadi pratijñā-anapekṣo virodhaḥ syāt, syāt parājaya-āśrayaḥ. pratijñā-adhikaraṇatve punas tat-prayogakṛta eva parājayo asya prastāva-upasaṃhāra-avasānatvāt. vyarthaṃ virodha-udbhāvanaṃ parājita-parājaya-abhāvād bhasmīkṛta-prajvalana-vat.

[31,17]

ye tu kecid vicāra-prasaṅgeṣv ekatra sādhye bahavo hetava ucyante, teṣāṃ vikalpena tat-sādhya-sādhanāya vṛtteḥ sāmarthyam anyathā dvitīyasya vaiyarthyāt. yadi hi tatra apy eka-prayogam antareṇa aparasya prayogo na sambhavet, na tadā dvitīyasya kaścit sādhana-arthaḥ pratīta-pratipādana- abhāvāt. tasmān na pratijñāyāḥ sva-vacana-virodho nāma kiṃcit nigrahasthānam.

[32,01]

na ca na asty ātmā ity atra kaścit pratijñā-virodhaḥ na asty ātma- śabdārthasya bhāva-upādānatva-niṣedhāt. śabdārtha-niṣedhe hi virodhaḥ syāt. na ca sva-lakṣaṇaṃ śabdārtha iti.

[32,04]

yaḥ punaḥ pratijñayā bādhanād hetu-virodha uktaḥ, yathā sarvaṃ pṛthak samūhe bhāva-śabda-prayogād iti, na atra pratijñāyāḥ prayogaḥ, na api hetor, yena virodhaḥ syāt, kiṃ tarhi pratipādita-artha-upadarśanena upasaṃhāra-vacanam etasmāt.

[32,09]

anyair eva hetubhiḥ śabdasya eka-viśeṣa-anabhidhānam aneka-artha-sāmānya-abhidhānaṃ ca pratipādya sarvasya śabdārthasya nānā-artha-rūpatayā eka-vastu-viśeṣa-svabhāvatā-bhāvam upadarśayan śabdārtham adhikṛtya sarvaṃ pṛthag iti brūyāt. etena tad-virodhaḥ pratyuktaḥ. dṛṣṭānta-upadarśanaṃ ca etad anityaḥ śabdaḥ kṛtaka-anityatvād iti yathā. kvacid arthe vipratipattau prasiddham aneka-artha-sāmānye śabda- prayogam upadarśya pratipādita-vipratipatti-sthānaḥ sāmānyena upasaṃharati sarvaṃ pṛthag iti.

[32,18]

yadi dṛṣṭānta-prayogaḥ kiṃ ṛjunā eva tat-prayoga-krameṇa na prayukto vipratipatti-viṣayaś ca kiṃ na darśita iti cet, na, samāsa-nirdeśāt, evam api prayoga-darśanād asādhana-vākyatvāc ca. ata eva na pratijñayā hetor bādhanam. na ca ekam eva kiṃcin na asti iti bruvāṇaḥ kaścit tat-samuccaya-rūpam ekaṃ samūham icchati yena virodhaḥ syāt.

[33,01]

yo api yugapat ṣaṭkena yogād ity ādinā paramāṇor bhedam āha, na tasya apy ekaḥ samuccaya-rūpaḥ sādhayitum iṣṭaḥ, kiṃ tarhy abhāva eva eka-aneka-pratiṣedhāt. ataḥ so api na samūhas tasya iṣṭaḥ, na tatra śabda iti na virodhaḥ.

[33,05]

na virodho ayaṃ pūrvakāt pratijñā-hetu-virodhāt bhidyate yena pṛthag ucyeta. tatra hetu-pratijñāyor bādhanam, iha pratijñayā hetor ity asti bheda iti cet, artha-virodhe hi hetu- pratijñayor bādhyabādakabhāvaḥ syāt. sarvo artha-virodho dviṣv ity api parasparaṃ bādhakam eka-artha-saṃnidhāv apara- arthā-sambhavāt. tatra hetu-pratijñāyoḥ pṛthag bādha-udāharaṇayor na kaścid artha-bhedaḥ.

[33,12]

api ca ayaṃ viruddho aviruddho vā sati hetu-prayoge vyadhikaraṇatvād asiddha ity asiddhatā hetor nigrahasthānam.

[33,14]

sa khalu ucyamāna eva atad-dharmatayā pratīto vaktuḥ parājayam ānayati. parājite tasmiṃs tad-artha-virodha-cintayā na kiṃcit.

[33,17]

api ca sarvatra ayaṃ pratijñā-hetvor virodhaḥ sambhavan na dvayīṃ hetu-doṣa-jātim atipatati viruddhatām asiddhatāṃ ca iti. viruddhatā siddhe hetor dharmiṇi bhāve sādhya-dharma- viparyaya eva bhāvena pratijñā-artha-virodhāt. asiddhatā punar dharmiṇi pratijñātā-artha-siddhau viruddhayoḥ svabhāvayor ekatra asambhavāt. na ca anyathā virodhaḥ.

[34,03]

asiddhe dharmi-svabhāve abhihitayor hetu-pratijñā-arthayor virodhād virodha-sambhava iti cet, apramāṇa-yoge tu ubhayor dharmiṇi saṃśayaḥ. tathā sati hetor dharmiṇi bhāva-saṃśaye asiddhatā eva hetu-doṣa ity asiddha-viruddhābhyām anyo na pratijñayā virodho nāma parājaya-hetuḥ. asiddha-viruddhe ca hetv-ābhāsa-vacanād eva ukta iti na pṛthak pratijñā-virodho vaktavya iti.

[34,10]

ubhaya-āśritatvād virodhasya vivakṣāto anyatara-nirdeśa iti cet, syād etat pratijñā-hetvor virodha iti pratijñā-hetū āśritya ubhaya-āśrayo bhavati. tatra yadā pratijñā-virodho vivakṣitas tadā pratijñā-virodha iti ucyate, yadā pratijñāyā hetor vā virodhas tadā viruddho hetur iti. ataḥ pratijñā-virodho hetu-virodho vā ity adoṣaḥ. tatra hetor udāharaṇaṃ nityaḥ śabda utpatti-dharmakatvād iti, pratijñā-virodhasya na asty ātmā iti, pratijñā-hetvoḥ paraspara-viruddha-udāharaṇaṃ guṇa-vyatiriktam ity ādi, pratijñayā hetu-virodha-udāharaṇaṃ na asty eko bhāva ity ādikam iti.

[34,20]

na, sarvatra hetv-apekṣasya virodhasya hetv-ābhāsa-anatikramāt, yathoktaṃ prāk. anapekṣe ca kevale svataḥ pratijñā- virodhe vivakṣite pratijñā-hetvor virodha iti hetu-grahaṇam asambaddham. na ca utpatti-dharmakatvān nityam ity atra api hetu-virodho yuktaḥ, pratijñayā hi hetor bādhane hetu-virodhaḥ. iha tu hetunā pratijñā bādhyata iti pratijñā-virodho yuktaḥ, ubhaya-āśraye api virodhe bādhyamāna-vivakṣayā tad-virodha-vyavasthāpanāt.

[35,06]

yad apy uktam etena pratijñayā dṛṣṭānta-virodha-ādayo api vaktavyā bhaṇḍālekhyanyāyena iti tatra api pakṣīkṛta-dharma- viparyayavati dṛṣṭānte virodhaḥ syāt. viruddhe ca dṛṣṭānte yadi pakṣa-dharmasya vṛttir ananya-sādhāraṇā prasādhyate, viruddhas tadā hetv-ābhāsaḥ sādhāraṇāyām aprasādhite vā tad-vṛtti-niyame anaikāntikaḥ avṛttau vā asādhāraṇaḥ.

[35,12]

viruddha-dṛṣṭāntā-vṛttau viparyaya-vṛttau ca hetor na kaścid hetu-doṣo dṛṣṭānta-virodhaś ca pratijñayā iti cet, na, tad api saṃśaya-hetutva-anativṛtter dṛṣṭānta-virodho hi pratijñāyāḥ sādharmye doṣaḥ, na vaidharmye, abhimatatvāt.

[35,16]

sādharmya-dṛṣṭānte ca viparīta-dharmavati vastutaḥ sādhyā-avyabhicāre api hetor na avyabhicāra-dharmatā śakyā darśayitum iti na apradarśita-avinābhāva-sambaddhād hetor niścayaḥ.

[35,19]

tan na pratijñayā dṛṣṭānta-virodho hetv-ābhāsān ativartate.

[36,01]

ubhayathā api doṣo astv iti cet, na na hetu-doṣasya prāk prasaṅgena parājitasya doṣa-antara-anapekṣaṇāt. viśeṣeṇa sādhana- avayava-anukrama-niyama-vādina udāharaṇa-sādharmyaṃ hetu-lakṣaṇaṃ viruddhe dṛṣṭānte na sambhavati iti prāk prayuktasya hetor doṣeṇa parājaya iti na uttara-dṛṣṭānta-apekṣayā virodhaś cintām arhati. hetor api dṛṣṭānta-virodhe asādhāraṇatvaṃ viruddhatvaṃ vā vaidharmye yadi vṛttiḥ syāt.

[36,08]

pramāṇa-virodhe tu hetoḥ, yathā na dahano agniḥ śaityād ity ādi hy asiddho hetv-ābhāsaḥ.

[36,10]

pratijñāyāḥ pramāṇa-virodhaḥ sva-vacana-virodhena vyākhyāta iti.

[36,12]

sarva ete sādhana-virodhā hetv-ābhāseṣv eva antarbhavanti iti hetv-ābhāsa-vacanena eva uktāḥ.

[36,14]

yat tu viruddham uttaraṃ para-pakṣe sva-siddhena gotva-ādinā anaikāntika-codanā iti, tad asambaddham eva. yadi hi sva-siddhena gotva-ādinā parasya vyabhicāra-siddhim ākāṅkṣeta, tasya tat sva-pakṣa-viruddhaṃ na abhimatam iti virodha yujyeta. sa hi svayaṃ pratipanne gotve hetu-vṛtteḥ saṃśayāno apratipattim ātmanas tathā khyāpayati. sa ca hetuḥ saty asati gotve aprasādhita-sādhana-sāmarthyaḥ saṃśaya-hetutvād anaikāntika eva prasādhite tu sāmarthye gotve avṛttyā hetor na saṃśaya eva sarva-saṃśaya-prakārāṇāṃ parihāreṇa samarthanāt. etena sva-pakṣa-anapekṣa-hetu-prayogasya anaikāntikatā vyākhyātā, so api sva-abhimata-nitya-gotva- vṛttiṃ hetum anityatve bruvāṇo asamarthita-sādhana-aṅgatayā saṃśaya-hetum eva āha iti. yat punar uktam ubhaya-pakṣa-sampratipannena vastunā anaikāntika-codanā iti, tatra apy avaśyaṃ saṃśaya-hetutva-mukhena eva anaikāntiko vaktavyaḥ. tad asamarthite anyatra api tulyam iti na ubhaya-siddha-itarayor anaikāntikatvair viśeṣaḥ.

[37,11]

yad apy uktam, dṛṣṭānta-ābhāsa-hetv-ābhāsa-pūrvakatvāt tad-abhidhānena eva uktā iti na pṛthag nigrahasthāneṣu uktā iti, tad apy avayava-antara-vādino ayuktam. yo avayava-antaraṃ dṛṣṭāntaṃ hetor āha, tasya na hetv-ābhāsa-uktyā dṛṣṭānta-ābhāsa- uktir vyāpyā tad-vacanena gamyamānasya tasmāt sādhana-antara- abhāva-prasaṅgāt. dṛṣṭānta-ābhāsānāṃ hetv-ābhāseṣv antarbhāvād dṛṣṭāntasya api hetāv antarbhāva iṣṭo bhavati, tathā ca na dṛṣṭāntaḥ pṛthak sādhana-avayavaḥ syād apṛthag-vṛtteḥ.

[37,19]

yo dṛṣṭānta-sādhyo arthas tasya hetāv antarbhāvād hetunā eva sādhita iti na dṛṣṭāntasya pṛthak kiṃcit sāmarthyam.

[38,01]

api ca na kiṃcit pūrva-pakṣa-vādino hetv-ābhāsa-asaṃsparṣe nyāyyaṃ nigrahasthānam asti iti tat-sambandhīni sarvāṇy eva hetv-ābhāsa-vacanena eva uktāni iti na pṛthag vācyāni syuḥ. artha-antara-gamana-āder api hetor asāmarthya eva sambhavāt.

[38,05]

na hi samarthe hetau sādhye ca siddhe artha-antara- gamanaṃ kaścid ārabhate asamarthasya mithyā-pravṛtter iti.

{2.4}

[38,07]

pakṣa-pratiṣedhe pratijñātā-artha-apanayanaṃ pratijñā-saṃnyāsaḥ.

[38,08]

yaḥ pratijñātam artham anityaḥ śabda aindriyakatvād iti sāmānya-vṛttyā hetor vyabhicāra-pradarśanena pratiṣedhe kṛte ka evam āha anityaḥ śabda iti parityajati, tasya pratijñā-saṃnyāso nāma nigrahasthānam iti.

[38,12]

atra api yady udbhāvite api hetor vyabhicāre na sa pakṣaṃ parityajati, kiṃ na gṛhyeta. nigṛhīta eva hetv-ābhāsa-abhidhānād iti cet, kim idānīm uttara-pratijñā-sāṃnyāsa-apekṣayā, tasya tad eva adyaṃ nigrahasthānam iti kim anyair aśakya-paricchedaiḥ klība-pralāpa-ceṣṭitair. upanyastair evaṃ hy atiprasaṅgaḥ syāt. pakṣa-pratiṣedhe tūṣṇīṃbhavatas tūṣṇīṃbhāvo nāma nigrahasthānam prapalāyamānasya prapalāyitvaṃ nāma nigrahasthānam ity evam ādy api vācyaṃ syāt, tasmād etad apy asambaddham iti.

{2.5}

[39,03]

aviśeṣa-ukte hetau pratiṣiddhe viśeṣam icchato hetv-antaram.

[39,04]

nidarśanam eka-prakṛti idaṃ vyaktaṃ parimāṇāt, mṛt-pūrvakāṇāṃ śarāva-prabhṛtīnāṃ dṛṣṭaṃ parimāṇam iti. asya vyabhicāreṇa pratyavasthānaṃ nānā-prākṛtīnām eka-prakṛtīnāṃ ca dṛṣṭaṃ parimāṇam iti evaṃ pratyavasthita āha eka-prakṛti-samanvaya-vikārāṇāṃ parimāṇa-darśanāt sukha- duḥkha-moha-samanvitaṃ hi idaṃ sarvaṃ vyaktaṃ parimitaṃ gṛhyate. tasya prakṛty-antara-rūpa-samanvaya-abhāve saty eka- prakṛtikatvam iti. tad idam aviśeṣa-ukte hetau pratiṣiddhe viśeṣaṃ bruvato hetv-antaraṃ bhavati. sati ca hetv-antara-bhāve pūrvasya hetor asādhakatvān nigrahasthānam.

[40,04]

atra api pūrvasya eva hetor anaikāntikasya abhidhānān nigṛhīte hetv-antara-cintā kva upayujyate. yadi prāk-sādhana-vādī hetum anaikāntikam uktvā datta-uttara-avasaraḥ, tena eva nigṛhyate adatta-uttara-avasaraḥ, hetv-antara-abhidhāne api na nigraham arhaty avirāmāt.

{2.6}

[40,09]

prakṛtād arthād apratibaddha-artham artha-antaram. yathokta- lakṣaṇe pakṣa-pratipakṣa-parigrahe hetutaḥ sādhya-siddhau prakṛtyāyāṃ kuryān nityaḥ śabdo asparśatvād iti hetuḥ, hetuś ca nāma hinoter dhātos tu-śabde pratyaye kṛd-antaṃ padam. padaṃ ca nāma-ākhyāta-upasarga-nipātā iti prastutya nāma-ādīni vyācaṣṭe, idam artha-antaraṃ nāma nigrahasthānam abhyupagata-artha-asaṃgatatvād iti.

[40,16]

nyāyyam etan nigrahasthānam, pūrva-uttara-pakṣa-vādinoḥ pratipādite doṣe prakṛtaṃ parityajyā-sādhana-aṅga-vacanam adoṣa-udbhāvanaṃ ca. sādhana-vādino hy upanyasta-sādhanasya samarthane kartavye tad akṛtvā aparasya prasaṅgena aprasaṅgena vā atan-nāntarīyakasya apy abhidhānaṃ parājaya- sthānam uttara-vādino api doṣa-udbhāvana-mātrād aparasya upakṣepa iti.

{2.7}

[41,05]

varṇa-krama-nirdeśavan nirarthakam. yathā nityaḥ śabdo jabagaḍadaśatvāt jhabhañghaḍhadhaṣavad iti. sādhana-anupādānān nigṛhyata iti.

[41,08]

idam apy asambaddham, na hi varṇa-krama-nirdeśād eva iṣṭa- artha-siddhāv ānarthakyam. yad eva kiṃcid asādhana-aṅgasya vacanam, tad eva anarthakaṃ sādhya-siddhy-upayogino abhidheyasya abhāvāt niṣprayojanatvāc ca iti, prakāra-viśeṣa-upādānam asambaddham.

[41,13]

vater upādānād adoṣa iti cet, syād etat, varṇa-krama-nirdeśavad iti vatir atra upāttaḥ so anyadā apy ananurūpaṃ gṛhnati ity adoṣa iti. na, artha-antara-āder nigrahasthānasya avacana-prasaṅgāt, evaṃ hi tā na pṛthag vācyā nirarthakena eva abhidhānāt.

[42,01]

na sādhya-siddhāv anarthakaṃ nirarthakam, yasya na eva kaścid arthaḥ, tan nirarthakam iṣṭam iti cet, yasya kasyacid avādino api hi nirarthaka-abhidhāne kiṃ na nigraho nigraha- nimitta-aviśeṣāt.

[42,05]

na, tasya iha aprastāvād iti cet, āyātam iha yo nirarthakaṃ bravīti, tasya tena eva nigraha iti. tat tulyaṃ sarvasya asādhana- aṅga-vādina iti. sa sarvo nirarthaka-abhidhāyy anena eva nigrahasthānena nigraha-arhaḥ.

[42,09]

na ca varṇa-krama-nirdeśaḥ sarvatra nirarthakaḥ kvacit- prakaraṇe tasya api arthavattvāt. tasmād atra eva asya ānarthakyāt nigrahasthānatvam.

[42,12]

api ca atyalpam idam ucyate varṇa-krama-nirdeśo nigrahasthānam iti kapola-vādita-kakṣa-ghaṭṭitakam ity evam-ādīnām api vācyatvāt.

{2.8}

[42,15]

parṣat-prativādibhyāṃ trir-abhihitam apy avijñātam avijñāta-artham. yad vākyaṃ parṣadā prativādinā ca trir-abhihitaṃ na vijñāyate śliṣṭa-śabdam apratīta-prayogam atidruta- uccāritam ity evam ādinā kāraṇena, tad avijñāta-artham, asāmarthya- saṃvaraṇāya prayuktaṃ nigrahasthānam iti.

[43,02]

na idaṃ nirarthakād bhidyate. sa yadi prakṛta-artha-sambaddhaṃ gamakam eva kuryāt, na asya-asāmarthyam, tatra jāḍyāt parṣad-ādayo na pratipadyanta iti na vidvān nigraham arhati.

[43,06]

parṣat-prajñām aparikalpya vacanān nigraha-arha eva iti cet, nyāyavādino jāḍyād uktam ajānan kiṃ na prativādī nigṛhyate.

[43,08]

jāḍyāt parṣad-āder avijñāta-pratipādana-asāmarthya iti vijetā na syān na nigraha-arhaḥ. asambaddha-abhidhāne nirarthakam eva iti na pṛthag avijñāta-arthaṃ nāma nigrahasthānam iti.

{2.9}

[43,11]

paurvāparya-ayogād apratibaddha-artham apārthakam. yatra anekasya padasya vākyasya vā paurvāparyeṇa yogo na asti ity asambaddha-arthatā gṛhyate, tat samudāya-arthasya apāyād apārthakam, daśa-dāḍima-ādi-vākya-vat.

[44,01]

idaṃ kila padānām asambandhād asambaddha-varṇān nirarthakāt pṛthag uktam nanv evam asambaddha-vākyam api pṛthag vācyaṃ syāt. na ubhaya-saṃgrahāt apārthakam, nirarthakasya api saṃgraha-prasaṅgāt. evaṃvidhāc ca viśeṣa- samāśrayāt pṛthag-nigrahasthāna-lakṣaṇa-pralapane atiprasaṅgo apy uktaḥ. na ca saṃgraha-nirdeśe kiṃcid doṣaṃ paśyāmaḥ prabhede vā guṇa-antaram iti yat kiṃcid etat.

{2.10}

[44,08]

avayava-viparyāsa-vacanam aprāptakālam. pratijñā-ādīnāṃ yathālakṣaṇam artha-vaśāt kramaḥ, tatra avayavānāṃ viparyayeṇa abhidhānaṃ nigrahasthānam.

[44,11]

na, evam api siddher iti cet, na prayoga-apeta-śabda-tulyatvāt.

[44,12]

yathā gaur ity asya padasya arthe goṇī iti prayujyamānaṃ padaṃ kakuda-ādimantam arthaṃ pratipādyati iti. na śabda-anvākhyānaṃ vyartham, anena padena go-śabdam eva pratipadyate, go-śabdāt kakuda-ādimantam artham; tathā pratijñā-ādy- avayava-viparyayeṇa anupūrvīṃ pratipadyate, ānupūrvyā ca artham iti, tathā hi pūrvaṃ karma-upādīyate tataḥ karaṇaṃ mṛt-piṇḍa-ādikaṃ loka iti.

[45,03]

tad etad unmattasya unmatta-saṃvarṇanam iva prayoga-apeta- śabda-vad etad iti. yadi goṇī-śabdāt kakuda-ādimaty arthe pratītiḥ, śabda-anvākhyāna-prayatnena arthaṃ na paśyāmaḥ.

[45,07]

goṇī-śabdasya artha-pratipādane asāmarthyāt pratipādaka- vyutpatty-artham anvākhyānam iti cet, nanu goṇī-śabdād api loke pratītir dṛṣṭā. satyaṃ dṛṣṭā, na tu sākṣād ity uktam.

[45,10]

uktam etan na punar yuktam, strī-śūdrāṇām ubhaya-pratīter abhāvāt. yaḥ khalu ubhayaṃ vetti śabdam apaśabdaṃ ca, sa evaṃ pratipadyate. yas tu nakka-śabdaṃ mukka-śabdam eva vā vetti, na nāsā-śabdam, sa katham apaśabdāc śabdaṃ pratipadya tato arthaṃ pratipadyeta. dṛṣṭā ca anubhaya-vedino api pratītir iti na paraṃparayā pratītiḥ.

[45,16]

arthe asamarthasya śabde api pratīti-janana-asāmarthyāc ca.

[45,17]

na hy arthe api vācakatvaṃ nāma anyad eva tad-viṣaya-pratīti- jananāt. apaśabdaś cec śabde pratītiṃ janayet, artha eva kiṃ na janayati. na hy etasya arthāt kiṃcid bhayaṃ paśyāmo yena taṃ pariharet.

[46,01]

akṛta-samayasya śabde apy apratīti-jananāc ca. na hy ayam apaśabdaḥ śabde api svabhāvataḥ pratītiṃ janayaty adarśanāt, samayata eva tu janayet. samaya-vaśād vartamāno artha eva kiṃ na pravartate. evaṃ hi pratipatti-paraṃparā- pariśramaḥ parihṛto bhavati.

[46,06]

viparyaya-darśanāc ca. śabdād artham apratipadyamānāḥ apaśabdair eva bahulaṃ vyutpadyamānā loke dṛśyante iti vyarthaṃ śabda-anuśāsanam.

[46,09]

na vyarthaṃ saṃskṛta-śabda-vyutpatty-arthatvād iti cet, ko ayaṃ śabdānāṃ saṃskāraḥ. na hy eṣāṃ prajñā-bāhu-śruty- ādikaṃ saṃskāraṃ paśyāmaḥ, na apy eṣām ekāntena śravyatā, na apy artha-pratyāyane kaścid atiśayaḥ. na dharma-sādhanatā mithyā-vṛtti-codanebhyaḥ saṃskṛtebhyo apy adharma-utpatteḥ, anyebhyo api viparyaye dharma-utpatteḥ.

[46,15]

śabdasya suprayogād eva svarga-modana-ghoṣaṇā vacana-mātram.

[46,16]

na ca evaṃvidhān āgamān ādriyante yuktijñāḥ. na ca dāna-ādi-dharma-sādhana-codanā-śūnya-kevala-śabda-suprayogān naraka-pātam iti bruvāṇasya kasyacin mukhaṃ vakrībhavati.

[46,19]

tasmān na saṃskṛto nāma kaścic śabdaḥ.

[47,01]

śiṣṭa-prayogaḥ saṃskāra iti cet, ke śiṣṭāḥ. ye vidita-vedyatā- ādi-guṇa-yuktāḥ. kaḥ punar eṣāṃ guṇa-utkarṣa-anapekṣo alīka-nirbandhaḥ, yat te amūn eva śabdān prayuñjate, na aparān. na ca atra kaścic śabde parokṣaḥ sākṣī, yata idam evaṃ niścinumaḥ prayuñjate nāma śiṣṭāḥ. nanv evaṃ vayaṃ guṇa-atiśayam apaśyantaḥ saṃskāraṃ keṣāṃcic śabdānām anumanyāmahe, tad-anvākhyāna-yatnaṃ vā, guṇa-atiśaya-abhāvāt.

[47,08]

veda-rakṣa-ādikaṃ ca aprayojanam eva atat-samaya-sthāyinaḥ.

[47,09]

saty api guṇa atiśaye na karaṇīya eva anvākhyāne yatnaḥ tat-svabhāvasya anyato api siddheḥ, prākṛta-apabhraṃśa- dramiḍa-andhra-ādi-bhāṣā-vat. na hi pratideśaṃ bhāṣāṇāṃ kiṃcil lakṣaṇam asti, atha ca sampradāya-vaśāt tā lokas tathā eva pratipadyate tāsāṃ ca prayoga-bhraṃśam tathā saṃskṛtānāṃ śabdānāṃ pratītir bhaviṣyati iti. jaḍa-pravṛttir eva eṣā, yā śabdānāṃ lakṣaṇe pravṛttiḥ.

[48,01]

avayava-viparyaye api yadi teṣāṃ vacanānāṃ sambandha-pratītir, na viparyayo na apy artha-apratītiḥ sāmarthyāt. na hy atra kaścit samayaḥ pratyāyana-aviśeṣe apy evam eva avayavāḥ prayoktavyā iti.

[48,05]

sa eva teṣāṃ kramo yathāvasthitebhyo artha-pratītir bhavati iti na viparyayāt pratītiḥ, tata ānupūrvī-pratipattyā pratītir iti cet, na apratīyamāna-sambandhebhya ānupūrvī-pratipattiḥ.

[48,08]

yeṣāṃ śabdānāṃ kaścit sambandho jāyate, idam iha sambadhyata iti teṣu vidita-sambandheṣu kaḥ kasya pūrvo aparo vā kramaḥ, yena krameṇa vyavasthāpyeran. sambandha- pratipattau sa eva teṣāṃ kramaḥ, yo yathāvasthitānāṃ sambandhaḥ pratīyate. na hi vākyeṣu padānāṃ krama-niyamaḥ kaścit, yathā rājñaḥ puruṣaḥ, puruṣo rajña iti. yāvadbhiḥ padair artha-parisamāptiḥ tadā ekaṃ vākyam, yathā devadatta gām ānaya kṛṣṇām iti. atra padānāṃ yathākāmaṃ prayoge api na artha-pratītau viśeṣa iti kaścit krama-abhiniveśaḥ.

[48,18]

pratipāditaṃ ca pratijñā-vacana-antareṇa api yathārthaṃ pratītir bhaviṣyati iti pratīyamāna-arthasya ca śabdasya prayoge atiprasaṅgaḥ. pariśiṣṭeṣu ca sambandhaṃ pradarśya dharmiṇi bhāvaḥ pradarśyeta, dharmiṇi bhāvaṃ pradarśya sambandhaḥ pradarśyeta iti na niyamaḥ kaścid ubhayathā api pratīty-utpatter ity uktam. apratīyamāna-sambandheṣu ca padeṣu na tebhya ānupūrvyā api pratītir iti na idam apārthakād bhidyata iti; na aprāptakālaṃ pṛthag vācyaṃ syād iti.

{2.11}

[49,06]

hīnam anyatamena apy avayavena nyūnam. yasmin vākye pratijñā-ādīnām anyatamo avayavo na bhavati, tad vākyaṃ hīnaṃ sādhana-abhāve sādhyā-siddheḥ.

[49,09]

na pratijñā-nyūnaṃ hīnaṃ tad-abhāve pratīti-bhāvād iti pratipāditam. hīnam eva tat nyūnatāyām api nigrahād ity aparaḥ. yaḥ pratīyamāna-artham anarthakaṃ śabdaṃ prayuṅkte, sa nigraham arhet. na artha-upasaṃhitasya abhidhātā ity asamīkṣita-abhidhānam etat. ata eva ca pratijñāyā na sādhana- aṅga-bhāva iti.

{2.12}

[49,15]

hetu-udāharaṇa-adhikam adhikam. ekena kṛtatvād anyatarasya ānarthakyam iti tad etan niyama-abhyupagame veditavyam.

[49,18]

yatra eka-sādhana-vākya-prayoga-pūrvako vicāraḥ, tatra adhika- abhidhānam anarthakam iti nigrahasthānam. prapañca-kathāyāṃ tu na kaścid doṣo niyama-abhāvād iti.

{2.13}

[50,01]

śabdārthayoḥ punarvacanaṃ punaruktam anyatra anuvādāt.

[50,02]

śabda-punaruktam anityaḥ śabdo anityaḥ śabda iti.

[50,03]

artha-punaruktam anityaḥ śabdo nirodha-dharmako dhvāna iti.

[50,05]

atra na śabda-punaruktaṃ pṛthag vācyam artha-punarukta- vacanena eva gatatvāt. na hy artha-bhede śabda-sāmye api kaścid doṣaḥ, yathā hasati hasati svāminy uccai rudaty atiroditi / kṛta-parikaraṃ sveda-udgāraṃ pradhāvati dhāvati / guṇa-samuditaṃ doṣa-apetaṃ praṇindati nindati / dhana-lava-parikrītaṃ yantraṃ pranṛtyati nṛtyati //2// yathā vā yad yasmin bhavati bhavati, na bhavati na bhavati, tat tasya kāryam, itarat kāraṇam iti.

[50,14]

gamyamāna-arthaṃ punarvacanam api punaruktaṃ niyata- pada-prayoge sādhana-vākye yathā pratijñā-vacanam iti.

[50,16]

artha-punaruktena eva gata-arthatvāt na pṛthag vācyam. ayam api niyata-sādhana vākya eva doṣo vaktavyaḥ, na vistara-kathāyām, vyācakṣāṇo hi kadācid asamyak-śravaṇa-pratipatti- śaṅkayā sākṣi-prabhṛtīnāṃ punaḥ punar brūyāt, api na tatra kiṃcic chalam.

[50,21]

na aviṣayatvād iti cet, na ayaṃ gurur na śiṣya iti na yatnataḥ pratipādaniyaḥ, yena punaḥ punar ucyata iti punarvacane nigraha eva iti cet, na, sākṣiṇāṃ yatnena pratipādyatvāt, tad-apratipādane doṣa-abhidhānāt, pratipādyasya śiṣyatvāt, vijigīṣu-vāda-pratiṣedhatvāt, trir-abhidhāna-vacanāt, punarvacana- prasaṅge samaya-niyama-abhāvāc ca.

[51,06]

na ca idam adhikād bhidyata iti na pṛthag vācyam. viniyata- pada-prayoge hi sādhana-vākye ādhikya-doṣa iti punarvacane api gata-arthasya-ādhikyam eva padasya iti.

[51,09]

prapañca-kathāyām apy akathita-eka-artha-sādhana-adhikaraṇāyāṃ nānā-artha-sādhana-īpsāyāṃ nānā-sādhana-īpsāyāṃ vā śrotur hetv-ādi-bāhulasya punarvacanasya adoṣatvāt pratīta- pratyaya-abhāvād hetv-ādi-bāhulyaṃ vacana-bāhulyaṃ ca sādhana- doṣa iti ādhikya-punarvacanayos tulya-doṣa iti saṃgraha- vacanaṃ nyāyyam, doṣa-abhāvād eva guṇa-abhāvāt, evaṃprakārānāṃ bhedānāṃ vacane ca atiprasaṅgād ity uktam.

[51,16]

paryāya-śabda-kalpo hy aparo hetur eka-hetu-pratipādite viṣaye vartamānaḥ pratipādyasya viśeṣa-abhāvāt. arthaḥ punaḥ- pratipādanān na bhidyate.

[52,01]

yat punar uktam, anuvāde tv apunaruktaṃ śabda-abhyāsād artha-viśeṣa-upapatteḥ, yathā hetv-apadeśāt pratijñāyāḥ punarvacanaṃ nigamanam iti, pratijñāyā eva gamyamāna-arthāyā vacanaṃ punarvacanam, kiṃ punar asyāḥ punarvacanam ity ayuktaṃ nigamanam.

{2.14}

[52,06]

vijñātasya parṣadā trir-abhihitasya apratyuccāraṇam ananubhāṣaṇam.

[52,07]

vijñāta-vākya-arthasya parṣadā prativādinā trir-abhihitasya yad apratyuccāraṇam, tad ananubhāṣaṇaṃ nāma nigrahasthānam. apratyuccārayan kim-āśrayaṃ para- pakṣa-pratiṣedhaṃ kuryād iti.

[52,11]

uttareṇa avasānān na idaṃ nigrahasthānam iti cet, syād etat, uttareṇa guṇa-doṣavatā mūḍha-amūḍhatvaṃ gamyata iti kiṃ punar-uccāritena. asti hi kaścid uttare samartho na pratyuccāraṇe, na asau tāvatā nigraham arhed iti.

[52,15]

na, uttara-viṣaya-aparijñānāt. yady ayaṃ na pratyuccārayati, nirviṣayam uttaraṃ prasajyeta. atha uttaraṃ bravīti, kathaṃ na uccārayati. tad idaṃ vyāhatam ucyate, na uccārayaty uttaraṃ ca bravīti iti. apratijñānāc ca. na ca idaṃ pratijñāyate, pūrvaṃ sarvam uccārayitavyam, paścād uttaram abhidhātavyam iti, api tu yathākathaṃcid uttaraṃ vācyam, uttaraṃ ca āśraya-abhāve ayuktam iti, yuktam apratyuccāraṇaṃ nigrahasthānam iti.

[53,06]

yadi nāma vādī sva-sādhana-artha-vivaraṇa-vyājena prasaṅgād aparāparaṃ ghoṣayet, vivāda-āspadaṃ ca jijñāsitam artha-mātram uktvā pratijñā-ādiṣv artha-viśeṣa-paraṃparayā aparān arthān upakṣipya kathāṃ vistārayet, tac ca sarvaṃ yadā na anuvaktuṃ śaknuyāt, kas tasya vivāda-āśraya-artha-mātra-uttara- vacane sāmarthya-vighātaḥ, yena vādi-vacana-ananubhāṣaṇaṃ nigrahasthānam ucyate. tasmād sarvāṃ vādi-kathām ananubhāṣamāṇo na uttare asamarthaḥ.

[53,14]

yad-vacana-nāntarīyakā jijñāsita-artha-siddhiḥ yathā pakṣadharmatā- vyāpti-prasādhana-mātraṃ na tatra api prasaṅga-antara- upakṣepaḥ, tad avaśyaṃ sādhana-aṅga-viṣayatvād dūṣaṇena upadarśyata eva. tatra api na sarvaṃ prāg anukrameṇa uccāritavyam, paścād dūṣaṇaṃ vācyaṃ dvir uccāraṇa-prasaṅgāt. dūṣaṇa- viṣaya-upadarśana-arthe anubhāṣaṇe vādi-vacana-anukrama- ghoṣaṇaṃ vyartham iti, na kāryam eva dūṣayatā asya ayaṃ doṣa iti nāntarīyatvāt pratidoṣa-vacanaṃ viṣaya-upadarśanaṃ kriyata eva. na hi sarva-viṣaya-upadarśanaṃ kṛtvā yugapad doṣaḥ śakyate abhidhātum, pratyarthaṃ doṣa-bhedāt. tasmād yaṃ padārthaṃ dūṣayati, sa eva tad-dūṣaṇa-viṣayas tadā pradarśanīyo na aparaḥ, tad-dūṣaṇe apara-upadarśanasya asambhavāt.

[54,08]

tasmin dūṣite punar anyo artho apara-doṣa-viṣaya ity ayam anubhāṣaṇe dūṣaṇe ca nyāyaḥ. sakṛt-sarva-anubhāṣaṇe api doṣa-vacana-kāle punar viṣayaḥ pradarśanīya eva, apradarśite doṣasya vaktum aśakyatvāt. tathā ca dvir-anubhāṣaṇaṃ kṛtaṃ syāt. tatra prathamaṃ sarva-anukrama-anubhāṣaṇaṃ niṣprayojanam.

[54,13]

dūṣaṇa-vādinā dūṣaṇe vaktavye yan na tatra upayujyate, tasya abhidhānam adoṣa-udbhāvanaṃ dvir-uktiś ca iti sakṛt-sarva-anubhāṣaṇaṃ parājaya-adhikaraṇaṃ vācyam.

[54,16]

tathā astv iti cet, syād etat, uktaṃ etad artha-antaraṃ nigrahasthānam iti. tatra sādhane yataḥ kutaścit prasaṅga-ādinā anāntarīyakā-abhidhānaṃ vādino artha-antara-gamanam eva iti sa nigrahā-arhaḥ. na kaścit tat-kathita-kriyamāṇa-prasaṅgo na prasajyate. na api tat tasya anubhāṣaṇīyam, na ca idam apy asmābhir anujñāyate, sarvaṃ prāk sakṛd vaktavyaṃ paścād dūṣaṇam iti, kiṃ tu dūṣayatā avaśyaṃ viṣayo darśanīyo anyathā dūṣaṇā-vṛtter iti.

[55,01]

evaṃ tarhi na ananubhāṣaṇaṃ pṛthag nigrahasthānaṃ vācyam apratibhayā gatatvāt, uttarasya hy apratipattir apratibhā. na ca uttara-viṣayam apradarśayann uttaraṃ pratipattuṃ samarthaḥ. na hy anākṣipta-anuttara-pratipattikam ananubhāṣaṇam. tena ananubhāṣaṇasya vyāpikāyām apratibhāyāṃ vihitaṃ nigrahasthānatvam ananubhāṣaṇe labdham eva, gavi vihitam iva sāsnā-ādimattvaṃ bāhuleye api. tasmād apratibhā eva nigraha-adhikaraṇatvena vācyā, na ananubhāṣaṇam.

[55,10]

kaś ca ayaṃ samaya-niyamas trir-abhihitasya ananubhāṣaṇam iti. yadi tāvat para-pratipādana-arthā pravṛttiḥ, kiṃ trir abhidhīyeta. tathā tathā sa grāhaṇīyaḥ, yathā asya pratipattir bhavati. atha para-upatāpana-arthā, tadā api kiṃ trir abhidhīyate, sākṣiṇāṃ karṇe nivedya prativādī kaṣṭa-pratīta- druta-saṃkṣipta-ādibhir upadrotavyaḥ, yathā-uttara-pratipatti- vimūḍhas tūṣṇīṃbhavati. na hi para-upatāpana-krame kaścin nyāyaḥ, yena kaṣṭa-pratīta-prayoga-druta-uccāritāni nivāryante trir-abhidhānaṃ vā vidhīyate. na ca para-upatāpāya santaḥ pravartante śāstrāṇi vā praṇīyante ity ado vaktavyam. tasmāt tāvad vaktavyam, yāvad anena na gṛhītam, na trir eva. agrahaṇa- sāmarthye prāg eva paricchinna-sāmarthyena parihartavyaḥ parān anupratibodhyeti.

{2.15}

[56,08]

avijñātaṃ ca ajñānam. vijñātaṃ parṣadā prativādinā yad avijñātam, tad ajñānaṃ nāma nigrahasthānam. arthe khalv avijñāte na tasya pratiṣedhaṃ brūyād iti.

[56,11]

etad apy ananubhāṣaṇa-vad apratibhayā eva gamyatvād avācyam. yathā ananubhāṣaṇe apradarśita-viṣayatvād uttara- pratipattir aśakyā iti anuttara-pratipattyā eva nigrahasthānatvam uttara-viṣaya-pradarśana-prasaṅgam antareṇa anubhāṣaṇasya vaiyarthyāt, tathā-jñāne apy uttara-apratipattyā eva nigrahasthānatvam, ajānānaḥ katham uttaram uttara-viṣayaṃ ca uttaraṃ brūyād iti viṣaya-ajñānam uttara-jñānaṃ ca nigrahasthānam, anyathā-pratibhāyā nirviṣayatvāt. anavadhārita- artho hi na anubhāṣet. ananubhāṣamāṇo viṣayam apradarśya-uttaraṃ pratipattuṃ na śaknuyād iti uttaraṃ na pratipadyeta jñāta-uttara-tad-viṣayasya uttara-apratipatter asambhavāt.

[57,06]

ubhayam etad uttara-apratipatteḥ kāraṇam iti.

[57,07]

tad-abhāve pratipattir bhavaty eva iti tayoḥ pṛthag vacane apratibhāyāḥ ko viṣaya iti vaktavyam, nirviṣayatvād avācyā eva syāt.

[57,10]

na uttara-ajñānam ajñānaṃ kiṃ tarhi viṣaya-ajñānam, ajñāte hi viṣaye uttara-ajñānāt tan na pratipadyeta iti asti viṣayo apratibhāyā iti cet, evaṃ tarhy ananubhāṣaṇaṃ nirviṣayam, ajñānena ākṣepāt. na hi viṣayaṃ samyak pratipadyamānaḥ kaścin na anubhāṣeta iti na ananubhāṣaṇaṃ pṛthag vācyam.

[57,15]

uttara-ajñānasya ca ākṣepāt. viṣaya-ajñānena uttara-ajñānam apy ākṣiptam eva. na hi viṣayam ajānann uttaraṃ jānāti iti na eva apratibhāyā viṣayo asti.

[58,01]

jñāte api viṣaye punar uttara-ajñānam apratibhāyā viṣaya iti cet, evaṃ tarhi viṣaya-uttara-ajñānayor api prabhedān nigrahasthāna- antarāṇi vācyāni. yathā ajñānasya viṣaya-ajñānam uttara- ajñānam iti prabhedād asaty api guṇa-atiśaye nigrahasthāna- antara-vyavasthā kriyate, tathā ajñānayor api sarvā-jñāna-ardha- ajñānam ity ādi-prabhedān nigrahasthāna-antarāṇi kiṃ na ucyante.

[58,07]

na ca ubhayasya apy ajñānasya saṃgraha-vacane kaścid doṣaḥ, guṇas tu syād iti saṃgraha-vacanaṃ nyāyyam.

[58,09]

tasmād ananubhāṣaṇa-jñānayor apratibhā-viṣayatvān na pṛthag-vacanam. api ca na pūrva-uttara-vādino hetv-ābhāsā-pratibhābhyām anyan nigrahasthānaṃ nyāyyam asti tad-ubhaya- vacanena eva sarvam uktam iti. tad-ubhaya-ākṣipteṣu prabhedeṣu guṇa-atiśayam antareṇa vacana-āder atiprasaṅgād vyarthaḥ prapañca iti.

{2.16}

[58,15]

uttarasya apratipattir apratibhā. para-pakṣa-pratiṣedha uttaraṃ yadā na pratipadyate, tadā nigṛhīto vaktavyaḥ.

[58,17]

sādhana-vacana-anantaraṃ prativiṣayam uttare vyarthaṃ tad-ajñāna-krama-ghoṣaṇa-śloka-pāṭha-ādinā kālaṃ gamayan kartavya- apratipattyā nigraha-arha iti nyāyyaṃ nigrahasthānam iti.

{2.17}

[59,01]

kārya-vyāsaṅgāt kathā-vicchedo vikṣepaḥ. yat kiṃcit kartavyaṃ vyāsajya kathāṃ vicchinatti, idaṃ me karaṇīyaṃ parihīyate, asminn avasite paścāt kariṣyāmi, pratiśyāyakalā me kaṇṭhaṃ kṣiṇoty evam-ādinā kathāṃ vicchinatti, sa vikṣepo nāma nigrahasthānam. ekatara-nigraha-antāyāṃ kathāyāṃ svayam eva kathā-antaṃ pratipadyata iti.

[59,07]

idam apy yadi pūrvapakṣa-vādī kuryād vyāja-upakṣepa-mātreṇa, na punar bhūtasya tathāvidha-kathā-uparodhinaḥ kāryasya bhāve, tasya sva-sādhana-asāmarthya-paricchedād eva vikṣepaḥ syāt. tathā ca idam artha-antara-gamana eva antarbhaved asamartha-sādhana-abhidhānād hetv-ābhāseṣu vā. prakṛta- sādhana-asambaddha-pratipatteś ca nirarthaka-apārthakābhyāṃ na bhidyate. atiprasaṅgaś ca evaṃprakārāṇām asambaddha- sādhana-vākya-pratipatti-bhedānāṃ pṛthag-nigrahasthāna- vyavasthāpane proktaḥ. atha uttarapakṣa-vādy evaṃ vikṣipet, tasya api sādhana-anantaram uttare pratipattavye tad-apratipattyā vikṣepa-pratipattir apratibhāyām artha-antare vā antarbhavati.

[60,06]

nanu na avaśyaṃ sādhana-dūṣaṇābhyām eva sarvasya pratipattir, yena sarvā vādi-prativādinor na asamyak-pratipattir hetv-ābhāseṣv apratibhāyāṃ vā antarbhavet. bhavati hi anibaddhena api kathā-prapañcena vivāda iti, na, asambhavāt.

[60,10]

ekatra adhikaraṇe viruddha-abhyupagamayor vivādaḥ syāt, aviruddha- abhyupagamayor anabhyupagamayor vā vivādā-abhāvāt.

[60,12]

tatra avaśyam ekasya prāg-vacana-pravṛttiḥ, yugapat-pravṛttau paraspara-vacana-śravaṇa-avadhāraṇa-uttarāṇām asambhavena pravṛtti-vaiphalyāt svastha-ātmanām apravṛtteḥ. tena ca sva-upagama-upanyāse avaśyaṃ sādhanaṃ vaktavyam, anyathā pareṣām apratipatteḥ, apareṇa ca tat-sambandhi dūṣaṇam.

[60,17]

ubhayor asamyak-pratipattau hetv-ābhāsa-apratibhayoḥ prasaṅga iti. sarvo nyāya-pravṛttaḥ pūrva-uttarapakṣa-upanyāso dvayaṃ na atipatati. etena eva vitaṇḍā pratyukta-abhyupagama- abhāve vivāda-abhāvāt.

[61,03]

yadā tarhy abhyupagamya vādaṃ viphalatayā na kiṃcid vakti, anyad vā yat kiṃcit pralapati, tadā kathaṃ hetv-ābhāsa- antarbhāvaḥ. asamarthita-sādhana-abhidhāna evam uktam, anabhidhāna anya-abhidhānayor api parājaya eva ity uktam abhyupagamya vādam asādhana-aṅga-vacanāt. etena adhikasya punaruktasya ca pratijñā-āder vacanasya ca nigrahasthānatvaṃ vyākhyātam, tad api hi pratipādita-artha-viparyayatvāt sādhana-sāmarthya-anabhidhānam. apratīta-pratyayatayā lakṣaṇāt sādhanasya asādhana-aṅga-vacanam iti nigrahasthānam iti.

{2.18}

[61,13]

sva-pakṣa-doṣa-abhyupagamāt para-pakṣe doṣa-prasaṅgo mata-anujñā.

[61,14]

yaḥ pareṇa coditaṃ doṣam anuddhṛtya, bhavato apy ayaṃ doṣa iti bravīti, yathā bhavāṃś cauraḥ puruṣatvād ity ukte sa taṃ pratibrūyād, bhavān api iti, sa sva-pakṣe doṣa-abhyupagamāt para-pakṣe taṃ doṣaṃ prasañjayan para-matam anujānāti iti mata-anujñā nigrahasthānam iti.

[62,01]

atra api yadi puruṣatvāc cauro bhavān api syāt, na ca bhavatā ātmā evam iṣṭaḥ, tasmān na ayaṃ caurya-hetur iti yady ayam abhiprāyaḥ, tadā na kaścid doṣaḥ, anabhimate tad-ātmani cauratvena hetu-darśanena dūṣaṇāt.

[62,05]

prasaṅgam antareṇa āñjasena eva kiṃ na vyabhicārita iti cet, yat kiṃcid etat, santi hy evaṃprakārā api vyavahārā loka iti. atha tad upakṣepam abhyupagacchaty eva, tadā apy asāv uttara-apratipattyā eva tat-sādhane nigraha-arhaḥ, na aparatra sva-doṣa-upakṣepāt, tat-sādhana-nirdoṣatāyāṃ hi tad-abhyupagama eva uttara-apratipattir iti tāvatā eva pūrvam āpanna-nigrahasya para-doṣa-upakṣepasya anapekṣaṇīyatvād iti.

{2.19}

[62,12]

nigraha-prāptasya anigrahaḥ paryanuyojya-upekṣaṇam. paryanuyojyo nāma nigraha-upapattyā codanīyaḥ, tasya upekṣaṇaṃ nigraha-prāpto asi ity ananuyogaḥ. etac ca kasya parājaya ity anuyuktayā parṣadā vaktavyam, na khalu nigraha-prāptaḥ sva-kaupīnaṃ vivṛṇuyād iti.

[63,01]

atra api yadi sādhana-vādinaṃ nigraha-prāptam uttara-vādī na paryanuyuṅkte, apratibhā eva asya uttara-apratipatter iti na paryanuyojya-upekṣaṇaṃ pṛthag nigrahasthānam. nyāya-cintāyāṃ punar na dvayor ekasya api atra jaya-parājayau, sādhana- ābhāsena artha-apratipādanād bhūta-doṣa-anabhidhānāc ca.

[63,06]

atha kaṃcid doṣam udbhāvayati, kaṃcin na, na tadā nigraham arhati uttara-pratipatteḥ.

[63,08]

arhati eva, sarvasya sato doṣasya anudbhāvanād iti cet, na santa iti kṛtvā sarve doṣā avaśyaṃ vaktavyā avacane vā nigrahaḥ, ekena api tat-sādhana-vighātād, eka-sādhana-vacana-vat.

[63,11]

yathā ekasya artha-asya aneka-sādhana-sadbhāve apy ekena eva tat-siddher na sarva-upādānam iti na doṣam udbhāvayann eva aparasya anudbhāvanān nigraha-arhaḥ.

[64,01]

atha pūrva-pakṣa-vādī uttara-pakṣa-vādinaṃ nigraha-prāptaṃ na nigṛhṇāti, tadā tayor nyāyena na ekasya api pūrvavaj jaya-parājayau. doṣa-ābhāsaṃ brūvānam uttara-vādinaṃ sva- sādhanād anutsārayato asamarthita-sādhana-aṅgatvān na jayo vādinaḥ, sarva-doṣa-asambhava-pradarśanena sādhana-aṅga-samarthanāt.

[64,06]

na apy uttara-vādino bhūta-doṣa-apratipādanāt. tasmād evam api na paryanuyojya-upekṣaṇaṃ nāma parājaya-adhikaraṇam iti.

{2.20}

[64,09]

asthāne nigrahasthāna-anuyogo niranuyojya-anuyogaḥ. nigrahasthāna- lakṣaṇasya mithyā-avasāyād anigrahasthāne nigṛhīto asi iti bruvan niranuyojya-anuyogān nigṛhīto vaktavyaḥ.

[64,12]

atra api yadi tat-sādhana-vādinam abhūtair doṣair uttara- vādy abhiyuñjīta, so asthāne nirdoṣe nigrahasthānasya abhiyokta- udbhāvayitā na bhavati. tathā ca bhūta-doṣa-udbhāvana-lakṣaṇasya uttarasya apratipatter itareṇa uttara-ābhāsatve pratipādite apratibhayā eva nigṛhīta iti na idam ato nigrahasthāna-antaram.

[65,03]

atha uttara-vādinaṃ bhūtaṃ sādhana-doṣam udbhāvayantam aparo doṣa-ābhāsa-vacanena abhiyuñjīta, tasya tena bhūta- doṣatve pratipādite sādhana-ābhāsa-vacanena eva nigṛhīta iti.

[65,06]

evam api na idaṃ hetv-ābhāsebhyo bhidyate. avaśyaṃ hi viṣaya-antara-vyāpty-arthaṃ hetv-ābhāsā nigrahasthānatvena vaktavyāḥ, tad-uktāv apara-uktir anarthakā iti.

{2.21}

[65,09]

siddhāntam abhyupetyā-niyamāt kathā-prasaṅgo apasiddhāntaḥ.

[65,10]

kasyacid arthasya tathābhāvaṃ pratijñāya pratijñātā- artha-viparyayāt kathā-prasaṅgaṃ kurvato apasiddhānto vijñeyaḥ. yathā na sato vināśaḥ, na asad utpadyata iti siddhāntam abhyupetya pakṣam avasthāpayati ekā prakṛtir vyaktasya, vikārāṇām anvaya-darśanāt. mṛd-anvayānāṃ śarāva- ādīnāṃ dṛṣṭam eka-prakṛtikatvam, tathā ca ayaṃ vyakta- bhedas sukha-duḥkha-moha-samanvito gṛhyate. tat sukha-ādibhir eka-prakṛtir iti. sa evam uktavān paryanuyujyate.

[66,03]

atha prakṛtir vikāra iti kathaṃ lakṣayitavyam iti. yasya avasthitasya dharma-antara-nivṛttau dharma-antaraṃ pravartate, sā prakṛtiḥ. yat tad dharma-antaram, sa vikāra iti. so ayaṃ prakṛta-artha-viparyayād aniyamāt kathāṃ prasañjayati. pratijñātaṃ khalv anena na asad āvirbhavati, na sat tirobhavati iti.

[66,08]

sad-asatoś ca tirobhāva-āvirbhāvāv antareṇa na kasyacit pravṛtty- uparamaḥ pravṛttir vā iti evaṃ pratyavasthito yadi sata ātma-hānam asataś ca ātma-lābham abhyupaiti, apasiddhānto bhavati atha na abhyupaiti, pakṣo asya na sidhyati iti.

[66,12]

iha api na kaścid aniyamāt kathā-prasaṅgaḥ yat tena upagatan na asad utpadyate, na sad vinaśyati iti tasya samarthanāya idam uktam eka-prakṛtikam idaṃ vyaktam anvaya-darśanād iti. tatra ekā prakṛtiḥ sukha-duḥkha-mohaḥ. tad-avibhakta- yonikam idaṃ vyaktaṃ tad-anvaya-darśanāt. vyaktasya tat-svabhāvatā-bheda-upalabdher iti sukha-ādīnām utpatti- vināśa-abhyupagama-abhāvāt sarvasya tad-ātmakasya na utpatti- vināśāv iti siddhaṃ bhavati. atra tad-uktasya hetor doṣam anudbhāvya vikāra-prakṛti-lakṣaṇaṃ pṛcchan svayam ayaṃ prakṛta-asambandhena aniyamāt kathāṃ pravartayati.

[67,06]

tatra idaṃ syād vācyam, vyaktaṃ nāma pravṛtti-nivṛtti- dharmakam, na tathā sukha-ādayaḥ. vyaktasya sukha-ādy-anvaye sukha-ādi-svabhāvatāyāṃ pravṛtti-nivṛtti-dharmatā-lakṣaṇam avahīyata iti. na tad-rahita-sukha-ādi-svabhāvatā vyakta-lakṣaṇa- virodhād iti. sukha-ādy-anvaya-darśanād ity asiddho hetur iti. evaṃ hi tasya sādhana-doṣa-udbhāvanena pakṣo dūṣito bhavati. so anupasaṃhṛtya sādhana-doṣaṃ kathāṃ pratānayan sva-doṣaṃ paratra upakṣipati.

[67,14]

ayam eva doṣo anena prakaraṇena ucyata iti cet, eṣa naimittikāṇāṃ viṣayaḥ, na lokaḥ śabdair apratipāditam arthaṃ pratipattuṃ samartha iti. sa eva ayaṃ bhaṇḍālekhyanyāyo atra api. yathoktena nyāyena pūrvakasya asādhana-aṅgasya asiddhasya hetor abhidhānād eva nigrahaḥ, na api aniyamāt kathā-prasaṅgād iti. idam api hetv-ābhāseṣv antarbhāvān na pṛthag vācyam.

{2.22}

[68,01]

hetv-ābhāsāś ca yathoktāḥ. hetv-ābhāsāś ca nigrahasthānāni.

[68,02]

kiṃ punar lakṣaṇa-antara-yogād hetv-ābhāsā nigrahasthāna- bhāvam āpadyante, yathā pramāṇāni prameyatvam ity ata āha yathokta-hetv-ābhāsa-lakṣaṇena eva nigrahasthāna- bhāva iti.

[68,06]

atra api yathoktaṃ kṛtvā cintyam eva, kiṃ te yathālakṣita-prabhedās tathā eva, āhosvid anyathā iti. tat tu cintyamānam iha atiprasajyata iti na pratanyate. hetv-ābhāsāś ca yathānyāyaṃ nigrahasthānam iti etāvan-mātram iṣṭam iti.

{03_04}

[68,10]

loke avidyā-timira-paṭala-ullekhanas tattva-dṛṣṭer vādanyāyaḥ parahita-ratair eṣa sadbhiḥ praṇītaḥ / tattva-ālokaṃ timirayati taṃ durvidagdho jano ayaṃ tasmād yatnaḥ kṛta iha mayā tat-samujjvālanāya //3// vādanyāyo nāma prakaraṇaṃ samāptam.
