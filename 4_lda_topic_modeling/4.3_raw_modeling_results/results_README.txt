Once ToPān is installed, simply place the folder pramana-nlp into the local directory ToPan/www/data. Then, with ToPān started (while in ToPān top directory, run R, issue the command "library(shiny)", then issue the command "runApp()") look on the LDAvis tab for the "pramana-nlp" entry in order to browse.
